■Caldera 買下SCO伺服器軟體部門及專業服務部門

馮文龍/編譯               Aug 3rd 2000

原本 Microsoft NT 與 Unix 爭霸的伺服器作業系統市場，自從 Linux加
入後，市場生態丕變。PC UNIX作業系統的領導廠商SCO八月二日宣
佈將旗下的伺服器軟體部門及專業服務部門售予  Caldera  System，而
Caldera是一家以OpenLinux作業系統及開發Linux上的商業解決方案為
主的公司。

------------------------------------------------------------------------------------------------

Caldera system以買下的 SCO 部門成立一家新的公司 Caldera Inc.，SCO
佔28%股權，Caldera System以股票交換一部份股權，合計市值約一億
兩千多萬美金。Caldera的主要股東Canopy集團同意貸款1800萬美元給
SCO，而SCO則繼續保有其Tarantella部門、SCO OpenServer的營業額及
其智慧財產權。

Caldera將取得SCO OpenServer的獨家代理，並將繼續對SCO OpenServer
及UnixWare的客戶提供技術服務。在結合了經過時間驗證的Unix及低
成本的Linux後，Caldera聲稱將提供業界第一套結合 Unix 及Linux的開
放網際網路平台﹙Open Internet Platform,OIP﹚，此平台將在8月23日兩
家公司共同主持的2000年論壇中發表。

從某個角度看這個事件，Linux收編了原本屬於同級市場的競爭對手，
而 SCO 只是將其逐漸被 Linux 所取代的市場讓出而已，但另一方面在
Caldera獲得SCO的重要技術後，將使得原來的Linux用戶得以開發高檔
的應用程式，諸如那些屬於高可用度的叢集軟體技術，如此將使Linux
的市場更廣深。一旦Linux的功能更加完備，產品更加成熟，其對微軟
NT及高階伺服器市場的滲透將更趨積極。

■易利信與紅帽成立家用通訊產品的策略聯盟

蕭喜那/編譯                                                                               Aug 2  2000

易利信與紅帽宣布將成立策略聯盟，以開發家用通訊所須的科技、產
品與服務。此項合作的第一個具體成果會是易利信無線螢幕電話，它
在去年二月的 CeBIT 上已經預先展示過，並且可望在今年年底時上市
。這項策略聯盟的目標在於拓展以 Linux 為基礎的資訊家電與行動設
備的商機。
 
------------------------------------------------------------------------------------------------

易利信與紅帽今天宣布成立策略聯盟，以共同開發全新的家用通訊消
費產品與服務。這些產品將結合現行的產業標準例如Java以及最新的
開放原始碼科技例如Embedded Red Hat Linux，同時也會支援現行與發
展中的各種標準例如寬頻網路以及 Bluetooth，易利信可說是 Bluetooth
的世界性領導廠商。

此項策略的第一個產品將是去年二月在 CeBIT 上預先展示過的易利信
無線螢幕電話，擬於今年年底上市。這項產品是一種電腦電話，由兩
部分組成：基地台與手提螢幕，二者都是在 Red Hat Linux的作業系統
下運作，並且與Bluetooth無線通訊科技相連。

另外，二家公司也將合作推出附加的網際網路服務，特別針對正積極
想要提供顧客新的、使用容易、與創新服務的電信與IASP業者。舉例
而言，將易利信的產品基座升級，可以使消費者使用最新的應用軟體
與內容，而不須煩惱科技的複雜程度。

易利信在此項策略聯盟中，也將與紅帽合作完成開放式的科技，例如
最新的Embeded Red Hat GNU開發工具，可在redhat.com上免費下載。
易利信與紅帽也正在建立一項新的機制，以製作全球資訊網的內容，
以及給予應用軟體認證，以供這些網際網路家電產品所用。

■免費的比較差？OS的新選擇─OpenBSD

張書銘/編譯                                                                              Aug 04 2000

在現今網際網路獨樹一幟的環境下，要選擇一套好的作業系統必須考
量愈來愈多的條件。單單追求穩定性已經不能滿足多樣化的需求，在
最小的耗費中，能夠求得最大的效益，才是使使用者得到滿足的關鍵
決勝點。而OpenBSD正符合了這樣的需求。

------------------------------------------------------------------------------------------------

過去傳統的刻板印象中，高階伺服器需要的作業系統總是不斷的訴求
著穩定性以及多人多工的能力。而提供商業用的UNIX系統廠商如 HP
和Sun等公司，也不斷的以它們的產品在表現上優於 Linux 及BSD系列
的免費系統為號召來做為其訴求。但在實例上，OpenBSD的優異表現
令人不得不重新考量這種優勢是否還能維持下去。

確實，在高階的硬體相容性和SMP的支援上 OpenBSD還不算完美，但
是OpenBSD在高網路負載的考驗下有超乎想像的高水準演出則是不爭
的事實。澳洲流量最大站台之一的 wiretapped.net 就是個很好的例子，
在其他站台紛紛為了大流量而不得不採用各種高階硬體和Sun server時
，wiretapped.net 平均每天高達 10GB 的流量，僅僅靠著一台K-6 450加
上128MB RAM和一台IDE的硬碟就可以搞定！

據wiretapped.net站台的站長表示，原本他們是採用Red Hat做為作業系
統，但是後來在迫於流量而不得不做全面升級動作時，考量到安全性
、網路的效率和系統負載等等因素後，嘗試了換用OpenBSD系統，而
OpenBSD也不負期望的滿足了各項需求。

然而，在目前台灣電腦普及而且寬頻網路風盛行的情況下，全球IP不
足的事實也成為令人注目的問題，雖然IPv6的規格已經漸漸成型，但
在過渡期間，一家之中多台電腦想同時上網成了一個難題。OpenBSD
追求簡單的系統配置設定使得要在家中建立起對外連線的區域網路不
再困難重重。因此OpenBSD在非商業伺服器的應用上也大有可為。這
也正說明了，OpenBSD追求降低整體系統價格並提升系統效能的方向
，也正符合了使用者真正的需求。

■Red Hat、SuSE 新版本套件及相關應用軟體更新相繼釋出

沈美君/編譯                                                                              Aug 01 2000

日前Red Hat 及 SuSE 兩大公司相繼發佈最新版本的套件，分別為紅帽
7.0 beta版本及 SuSE 7.0 Professional/Personal版本，而其他在Linux 上的
相關應用軟體或視窗管理員也陸續推出新版本或修正，對使用者來說
無非是一大福音。

------------------------------------------------------------------------------------------------

雖然Linux最新版測試核心﹙kernel 2.4﹚遲遲未問世，但各家發展Linux
套件的公司無不摩拳擦掌，致力發展或修正系統核心外的其他軟體和
套件。像歐洲SuSE Linux公司發佈的最新版本套件：SuSE 7.0，而紅帽
公司隨即也發佈 Red Hat 7.0 測試版，供使用者下載。

新版的 SuSE 7.0 分為 Professional 和 Personal兩種版本，主要用以區分
專業伺服器和個人桌上型的市場。Professional 版本提供 6 張 CD-ROM
和1張DVD，包括超過1500 種更新工具和軟體套件，提供企業完整的
網路伺服器解決方案，加強資料庫應用和支援4GB記憶體，也附上完
整的使用安裝手冊，SuSE 並使用 SuSE 新發展的最佳化自動安裝和調
校環境ALICE﹙Automatic Linux Installation and Configuration Environment
﹚，提升安裝和管理的效率。

SuSE  7.0  Personal 版本中有豐富的多媒體、影像處理、遊戲和網路相
關應用軟體，StarOffice  5.2 有強大的文書處理、試算表等功能，相當
適合個人桌上型使用。

紅帽  7.0 測試版本又被稱為「細條紋」﹙ Pinstripe ﹚，和之前的紅帽
 6.X 版比較起來，有不少相異或改進的地方，例如可相容於FHS 的套
件包裝、Apache 和 FTP 的預設目錄不再是 /home、 LDAP 和 Kerberos
的擴充整合、新一代函式庫 pango 和 Inti 預覽、支援 3D加速和 USB、
某些系統服務的改變﹙例如 inetd 將被xinetd所取代﹚。測試版本除了
在紅帽官方站可下載之外，各地也有映射站台可提供方便快速的下載
，但測試版有許多套件或原件因為版本太新，不是很穩定，比較適合
進階使用者來測試使用。

除了  Linux 套件的新版本發佈外，最近還有幾項新的發展。 SteelEye 
公司在近期也發佈了最新的 Linux 叢集技術﹙clustering﹚，以軟體整
合叢集的分散平行運算環境，確保Apache web server 能穩定的提供服
務。

另外，兩大圖形桌面整合環境 KDE 和 Gnome 也有新動作。KDE 日前
發佈了KDE2.0的測試版，除了更穩定、介面更完善以外，也有幾項新
的特點：將更容易建立檔案關連性、修正因動態記憶體配置而造成的 
memory leak、工作列上的對話盒，允許使用者直接使用命令模式。值
得一提的是，開放原始碼界的殺手級辦公室軟體  Koffice  完整整合於 
KDE2.0 之中，功能和效率都是得到好評的。

Gnome 中整合的檔案管理員 Nautilus，由 Eazel 公司所發展，讓使用者
完全以圖形介面來管理檔案及執行命令，Nautilus亦成為 Red Hat Linux
未來的預選使用項目之一。

■XFS值得我們等待

王唯農/編譯                                                                              Aug 04 2000

XFS團隊目前正對Linux的page cache做加強，使得XFS可以完整地整合
入Linux的核心。這工作即將完成，讓XFS可以單獨地在 page cache 上
同時去儲存檔案資訊與檔案系統的相關資訊。因為這技術是在核心層
級中發展，使得其他的檔案系統也可以利用它提升整體的系統性能。

------------------------------------------------------------------------------------------------

1994年SGI發表了一套在IRIX上新的日誌式檔案系統，叫做XFS，開發
此先進的檔案系統是為了要替代SGI 舊的EFS檔案系統。SGI 現在正和
其他的核心發展者合作將XFS整合進Linux，讓我們有比 ext2更好的檔
案系統可以使用。

日誌式的檔案系統的特色是會去記錄檔案系統的動作，因此當系統毀
損時，可以很快的做修復。這代表的是當系統電力突然中斷時，檔案
系統有記錄知道當電力中斷時的檔案系統狀況，它不用去把整個的檔
案系統做一次檢察，它只要檢察當時的記錄，就可以做修復了，節省
了很多的時間。

XFS廣泛地採用了B+ trees以取代傳統的線性檔案系統結構。B+ trees利
用了一種高效率的索引方法去將每筆目錄編為索引，管理檔案，使用
空間，並保持檔案的索引資訊。因此我們不必用大量的系統資源，就
可以達到快速的讀寫了。

XFS是一個64位元的檔案系統，也就是說，它可以支援一個很大很大
的檔案﹙9百萬terabytes﹚ 及更大的檔案系統﹙1千8百萬terabytes﹚。

現在，XFS團隊正對Linux的page cache做加強使得XFS可以完整地整合
入 Linux 的核心。這工作即將完成，讓 XFS可以單獨地在page cache上
同時去儲存檔案資訊與檔案系統的相關資訊。因為這技術是在核心層
級中發展，使得其他的檔案系統也可以利用它提升整體的系統性能。

■VA Linux 推出網路基礎及軟體發展服務

黃翔祺/編譯                                                                              Aug 07 2000            

開放系統主要的優勢，來自開放的模式讓更多的廠商能夠加入，破除
了硬體廠商以硬體設施作為軟體收費的方式，除了紅帽和SuSE積極投
入之外，VA Linux 進一步提供全方位專業的企業資訊系統建置及軟體
開發服務。

------------------------------------------------------------------------------------------------

開放系統主要的優勢，來自開放系統的讓更多的廠商能夠加入，並且
一起改進系統，如果運用得當，整個linux社群將可以一起尋找問題，
改進問題，可以持續進行產業創新。

linux 系統發展模式，破除了實體世界以硬體設施作為軟體收費的方式
，當消費者購買昇陽的硬體系統時，部分的費用可能是用來補貼昇陽
作業軟體的費用，另外，也破除微軟以軟體系統獨佔整體個人電腦產
業的情況，造成不合理的競爭。在這樣狀況下，系統廠商發展開放原
始碼系統來滿足顧客多樣化的需求，除了傳統的大廠：紅帽和SuSE外
，最近最積極投入的廠商就屬於 VA Linux 。

VA Linux 系統七號宣布成立，未來將拓展「專業服務部門」，主要提
供網路基礎開放原始碼的軟體發展服務，及專業管理顧問業務。此部
門在過去的三個月內網羅了 40 位工程師，包括幾位在 linux 界，享有
盛名的幾位天才技術人員，如：以開發圖像系統著稱的 Daryll Strauss
，推行 xfree86 計畫的David Dawes等人。

公司發言人表示，公司經過拓展後，將可以提供完整的企業軟體服務
，由企畫開始，一直到軟體開發，到系統建置等。透過專業軟體公司
，網際網路公司將可以更專注在其核心業務的經營上，推出更好的產
品。

VA linux 是產業界首屈一指的系統廠商，提供單一的系統授權服務，
也是軟體業界第一流的軟體開發廠商，VA Linux 本身經營SourceForge
開放原始碼中心，業界經營相同業務的還有 linux.com ，及Themes.org
。此外，由  VA linux 的實力及努力，被  fortune 雜誌、Upside 雜誌、
Red Herring 網站選為最值得投資的軟體廠商，並且在美國Nasdaq股市
上櫃。

Lutris Technologiese公司6月26日發表的Lutris Enhydra Professional 3.0，

及管理方式提供開放性原始碼社群及企業合作的新模式。

------------------------------------------------------------------------------------------------

Lutris Technologiese公司6月26日發表Lutris Enhydra Professional 3.0，是
該公司最新版的開放性原始碼Java/XML應用軟體伺服器，包含研製及
開發的整合套件，能建立部門及企業的Web應用軟體，跨平台整合所
有Web伺服器。

Lutris表示，面對行動商務時代的來臨，結合可接收Web內容的新平台
，對想藉由Web進行商業經營的企業，極為重要。Lutris為因應Web對
行動用戶上的應用層面日漸增加，為開發者提供新的基礎。

Enhydra 是一套業界最主要的 Java/XML，網際網路及無線應用軟體伺
服器，屬於開放性原始碼軟體，採有機及模組化設計，將介面設計、
企業邏輯、資料物件完全分離。該應用軟體能夠自所有JDBC API相容
的資料庫中存取資訊，提供IT部門建立分散式網路及無線應用軟體。

Lutris  Technologies 公司，為一家開放性原始碼企業軟體及服務公司，
提供研討會，及維護  Enhydra 開發社群基礎組織，該組織成員來自全
世界各處，採用  Enhydra 作為廣泛應用軟體的延伸基礎，是一個快速
成長的開發者團體。 Lutris  同時也是一家網際網路專業服務公司，為
Enhydra社群及跨足網際網路的企業提供策略及開發服務。

Enhydra最初由 Lutris Technologies 主導開發，採封閉原始碼。在去年1
月，Lutris將Enhydra的原始碼公開，並成立 enhydra組織監督未來開放
性原始碼  Enhydra  的開發過程。因此，比起其它開放性原始碼專案，
Enhydra 結合開放性原始碼社群及商業合作，具有結構穩固的管理基
礎。

雖然 Enhydra的核心授權採Free BSD模式，但其先進的XMLC元件，及
其它Enhydra開發的產品，則採取EPL﹙Enhydra Public License﹚授權，
不必開放原始程式碼。為了保障Enhydra的開發工作將回饋開發社群，
EPL 特別規定，任何 Enhydra 核心程式碼的更動，必須提交到 enhydra
組織，包含在Enhydra套件中以供下載。

Lutris Enhydra Professional 3.0是 Lutris產品家族的最新成員。Lutris所提
供的其它Enhydra產品則包括，目前已有的Lutris Enhydra Standard 及預
計年底發表的 Lutris  Enhydra  Enterprise。Lutris Enhydra Professional 3.0
的售價為499美元，並包含技術支援。

■SteelEye提供Apache使用者解決方案

吳政斌/編譯                                                                              Aug  3  2000

SteelEye最新的網頁伺服器解決軟體，結合了SteelEye於Linux平台的產
品LifeKeeper在內，並增加了許多新的功能，提供Apache使用者一個
容易使用的環境。

------------------------------------------------------------------------------------------------

SteelEye最新的網頁伺服器解決軟體﹙SteelEye  High  Available  Web 
Server  solution   for  Apache﹚，結合了  SteelEye  於  Linux  平台的產品
LifeKeeper。大量的解決方案幫助了網頁服務提供者或網路技術專家，
建立高效能的網頁伺服器，以供電子商務系統使用。

為了保證系統的正常運作，它整合了多重的偵錯機制以偵測和恢復軟
硬體錯誤，或應用在防止系統溢位及執行程式時失敗。此一方案的主
要功能包含了主動動態的組態設定，藉以消除不必要的閒置伺服器，
減輕系統的負荷量，並且隨時對管理者回報及紀錄。

值得注意的一點是，程式藉由Java的圖形介面來集中控制，讓使用者
在使用上相當的容易，免除了以往需要一步一步輸入指令的繁瑣工作
。它同時還提供了共享磁碟及本機磁碟的儲存配置系統，並維持兩者
資料的一致性，當其中一台出現無法修護的錯誤時，可以及時取出另
一份資料來替補。

此網頁伺服器解決軟體服務的對象可以小自一般的小型電子商務，大
至大型企業商務網站的規劃建構。因為所需的成本很低，加上經過簡
單訓練即能使用自如，它還可在RedHat 6.1及Caldera eServer 2.3平台上
使用，包含了LifeKeeper﹙for Linux 3.0版﹚及 Apache 應用軟體恢復工
具。

■ 採2.4核心的紅帽Linux 7.1推出

梁天佐/編譯
2001/04/17

紅帽原本在Linux美國市場佔有率最高，但因去年9月推出的 7.0版瑕
疵不少，傳出安裝、臭蟲與穩定性的問題，以致 7.0版銷售狀況不佳
，甚至讓SuSE大幅超越；現在紅帽正式公開7.1版，除了修改7.0的問
題、增加內容，紅帽還提出網路服務解決方案。

------

最知名的Linux套件發行商紅帽公司，日前發表最新版Linux作業系統
紅帽Linux 7.1，此版本最大的特色為採用最新的2.4核心，改進多處
理器效能，以及增加新的網路應用程式與服務。除強化針對企業伺服
級用戶的功能外，也提供加強版圖形介面的軟體安裝程式，以及更豐
富詳細的使用者說明。

由於以2.4為核心，紅帽Linux 7.1可使配備八個處理器的平台發揮最
大的效能，支援64G實體RAM記憶體，支援更多使用者或組群進行更多
處理，且支援USB 等通用傳輸方式，使IT人員能夠輕鬆應用相關設備
；標準安裝提供紅帽網路服務﹙Red Hat Network Software Manager
﹚，可自動提醒使用者更新、修正檔與新版軟體訊息，管理人員可藉
以了解網路上各個電腦的資訊，進行遠端安裝、移除與維修等工作。

安全性上，紅帽Linux 7.1以Heightened Security提供主動防護，連
接埠預設關閉，網路應用程式在不用時也會被關閉，同時也提供防火
牆操作螢幕介面與系統組態工具，使用者與管理人員可輕易設定安全
等級、BIND﹙DNS﹚、 阿帕契伺服器與列印設備；此外在桌面環境上
，使用者可選擇最新版KDE或GNOME，內含具有e-mail客端與新聞讀取
功能的 Mozilla瀏覽器。其他的附加的功能與配備還有：支援2D與3D
圖形的XFree86 4.0.3 、裝置熱抽換、兩百頁的自訂系統指南與 TUX
伺服器等，應有盡有。

雖然新版作業系統功能齊全，紅帽主要還是將重心置於中小型企業市
場，紅帽認為配合 7.1版所推出的訂閱服務，可讓中小型企業省卻雇
用大量IT人員的成本，享有開放原始碼軟體在成本與效能更新上的優
點；紅帽本身也可自諮詢、支援與相關服務中獲取公司合理利潤。紅
帽執行長Matthew Szulize說：「7.1與紅帽首次的網路訂閱服務系列
，輕鬆的讓組織很快了解開放原始碼運算能力的優點。」

紅帽Linux 7.1 已於紅帽網站開放下載，包裝版本分成標準版、豪華
版與專業版三種組合，預定4月24日在全球上市， 三種版本都含有生
產性應用程式、昇陽StarOffice辦公室套裝軟體、Loki遊戲軟體CD與
使用者指南，不同點在於售後服務範圍與同捆專業軟體種類，使用者
可視需求選購。

此外，紅帽Linux 7.1以SPECweb99標準對一、二、四與八顆處理器架
構的實測數據，相對於其他作業系統都保持第一，相較於 7.0，確實
有大幅度的改進。

■ 康柏「水星計劃」  以Linux取代Win CE

張書銘/編譯
2001/04/16

康柏推行的「水星計劃」，將以Linux取代 Win CE系統，開發全新手
持裝置PDA產品。在傳出預備上市的消息後，引起PDA使用者的熱烈關
切，但康柏表示，水星計劃目前仍在繼續研發中，上市日期未確定。

------

康柏公司在水星計劃上跨越微軟的藩籬，採用 Linux系統，同時與麻
省理工學院合作，共同開發手持裝置的解決方案。日前有消息傳出該
計劃已經完成，並將於秋季開始出貨，而引起 PDA玩家的熱烈討論，
不過康柏公司已經對外表示該傳言並不真實，水星計劃仍在研發中。

水星計劃﹙ Project Mercury﹚是開發PDA的產品，由 Windows CE轉
用Linux 系統，康柏公司將與麻省理工學院進行長達五年的合作研究
，開發代號稱為H21的解決方案。研究的第一步是以康柏iPAQ  H3600
系列彩色PDA為樣本，並使用嵌入式Linux作業系統。

此計劃繼承了iPAQ H3600系列設計有多個擴充槽的特性，具備優良擴
充性的特色，尤其儲存裝置的容量大幅增加，若使用 IBM硬碟則最大
儲存容量可達1GB。此外，Mercury BackPAQ內建數位相機，並安裝有
Linux以及X Window，可支援802.11b無線通訊標準。

康柏公司表示，第一套Mercury BackPAQ原型機雖然在1月中完成，但
外界傳言正式機種將在秋季完成的消息並不確實，因為這個計劃是康
柏與麻省理工學院首度合作，在研究上仍有必須溝通的地方，產品仍
在研發階段；此外，康柏公司另有Compaq  Itsy等計劃在進行，因此
Mercury BackPAQ的上市時間仍然是未知數。

■ 成大MP3事件引起國際注意

梁天佐/編譯
2001/04/16

台南地檢署突擊搜索成大學生宿舍，並扣留內有MP3 檔案學生電腦的
新聞不只在國內引起校園震撼並引起社會廣泛的討論；由於 Napster
事件尚未塵埃落定， MP3點對點﹙peer-to-peer﹚傳輸爭議又起，利
益團體有意藉此殺雞儆猴，引起相關社群熱烈的討論。

------

成大宿舍 MP3事件之後，各界對檢警搜查、智財侵權等問題議論紛紛
；國內校園則是風聲鶴唳，學生間相傳如何隱藏或臨時緊急銷毀 MP3
檔案、硬碟的對策。另一方面，「國際唱片業交流基金會」﹙IFPI，
International Federation Phonographics Industry﹚ 派出代表到
台南地檢署備案，有意對十四位電腦被查扣的學生提出侵權控告。

消息傳至國外，原本對Napster與點對點MP3音樂的討論與爭議的社群
與團體都將焦點集中在成大事件上， MP3本身的使用與合法性固然又
再度被討論，但許多評論對台灣檢警單憑一紙檢舉信函，便能夠號稱
合法大肆搜索、並查扣學生私人電腦感到不可思議。

IFPI與國內反仿冒團體藉此事件突顯國內盜版問題嚴重，但 MP3與其
他版權軟體所面臨的不僅是法律，而是環境與制度是否合理的問題；
音樂工業和商業電腦軟體一樣，都是由市場行銷公司主導，商業公司
向創作者買下音樂權利，然後以合約與法律限制音樂流傳的方法，以
確保商業利益，但當網路與科技足以替代中間商業公司所提供的服務
時，不合理的費用自然會被人唾棄，以法律來定義盜版與違法並不能
真正解決問題。

商業公司常以盜版使用軟體會扼殺創作者生計的說法強調版權的道德
性，其實並不完全正確，以實際的開放原始碼運作為例，開發人員或
組織也可以向使用者收取合理的費用，由於省卻過度的行銷、包裝與
宣傳，開發人員與使用者之間都能得到最大的利益。許多音樂商品大
部分的成本都花費在此，很難說服消費者心甘情願付費。

從時代的角度來看， MP3其實是一種進步，它能夠以更經濟、方便且
有效的方式解決人們對音樂的需要，商業公司應該記取當年從LP塑膠
大唱盤與錄音帶轉變成CD環境的經驗，從新思考合理的商業模式，因
為科技的進步勢將不斷顛覆既有的不合理現象，以法律只能治標，事
情的本質並不會改變。Richard M Stallman聲援MP3， 在其筆記型電
腦外就貼有「MP3 IS NOT A CRIME」。

以商業公司的定義而言，東亞地區盜版嚴重，同區包括台灣在內的幾
個國家都常被指責是海盜王國，不時受到由利益團體遊說美國政府所
施加的貿易報復法案威脅；台南地檢署轟動國際的查緝行動至少在視
聽上向利益團體宣示要遵守規則了，但全世界都知道，真正的問題並
沒有解決。

■ SPSS推出DEES 3.0  讓Linux資料分析能力更優異

張書銘/編譯
2001/04/16

SPSS Data Entry Enterprise Server 3.0﹙DEES﹚將在Linux平台上
推出。DEES是一套網際網路問卷調查工具，它提供了問卷設計、資料
收集與管理，以及資料統計分析等能力。SPSS公司將把最新版DESS軟
體移植到Linux平台，提供更多使用者進行分析行銷的工作。

------

目前應用電腦與網際網路進行統計分析工作已成為趨勢。以發展統計
分析軟體聞名的SPSS公司，為了加強各平台上的資料分析能力，特別
將公司最新 SPSS Data Entry Enterprise Server 3.0﹙DEES﹚移植
至Linux平台。

DEES是一套網際網路問卷調查工具，它提供了問卷設計、資料收集與
管理以及資料統計分析等能力。相較於傳統問卷，它可以透過網際網
路發布問卷，並直接收集填寫好的問卷資料，減少問卷會收的時間；
同時也無需再做資料輸入，可直接透過SPSS統計分析模組進行資料分
析，讓使用者掌握即時資訊。

在最新的DEES 3.0版當中，針對網路技術部分做了許多改良，包括：
HTML文件的輸出能力，讓使用者在設計網路問卷時能夠達到所見即所
得的效果；加強網路安全控管，提供 SSL安全機制，確保填寫問卷者
個人資料不致外洩；同時SPSS公司也設計了新的認證方式，避免重複
填寫問卷或者大量投票程式的干擾，確保DEES 3.0收集的資料能反映
出實際的意見。

SPSS公司表示，過去DEES只有Windows 版本，但現在許多研究機構和
企業都開始使用Linux 系統，因此有必要為這些使用者開發新版本。
新版DEES可支援的作業系統包括：Windows NT、Solaris 2.6 以上或
Linux 2.2.12以上。一般使用者若不需要應用較複雜的功能，則只需
訂購SPSS Data Entry Builder以及SPSS Data Entry Station即可。

■ 托瓦玆澄清對Mac OS X的批評

李孟諺/編譯
2001/04/17

即使是如托瓦茲這樣重量級的人物說錯話，電腦使用者仍會不留情面
地給予批評。托瓦茲在他最新的自傳「Just for fun」中，表達了個
人對Mac OS X的看法後，遭受到蘋果族的猛烈抨擊。而他本人事後則
認為是「媒體引用失當」，並澄清遭受誤解的部分。

------

托瓦茲在他的新自傳「Just for fun」當中，表達了自己對Mac OS X
採用的核心Mach的看法。但是被讀者認為書中的評論是在批評蘋果公
司的Mac OS X系統，因而遭到蘋果族的砲轟，但是他在最近一次的訪
問當中，對此事件做出了澄清。

究竟是什麼引起軒然大波呢？托瓦茲在自傳中有一段寫到他與蘋果公
司執行長Steve Jobs談論Mac OS X的過程。他寫到Jobs不清楚他個人
對Mac OS X的核心程式Mach評價不高，而想邀他加入該作業系統的開
發。但他個人認為Mach是「垃圾」﹙a piece of crap﹚。

就是這個「垃圾」說法讓托瓦茲近來飽受蘋果族的猛烈批評。不過他
在最近接受LinuxWorld.com專欄作家Joe Barr的訪談當中，澄清了這
項指控，並表達了不少他個人Mac OS X讚賞的地方。

在訪談當中托瓦茲表示，Mac OS X的優點全在核心程式之外，而且最
特別的是使用者介面的便利。不過在稱讚的內容當中，完全沒有提到
他個人評為「垃圾」的Mach；至於日前媒體報導他不喜歡Mac OS X的
新聞，托瓦茲表示是媒體「引用失當」。不過他依舊沒有收回書中認
為Mach是「垃圾」的說法。

除了對Mach發表意見之外，書中也提到對蘋果公司執行長Steve Jobs
的看法。托瓦茲在書中表示，當初與Steve面談時，Steve似乎認為托
瓦茲如果要往桌面環境發展，就必須與蘋果合作。而托瓦茲認為，蘋
果公司沒有任何有趣的地方吸引他。

雖然托瓦茲對垃圾說提出了澄清，但是蘋果族卻已經被他惹惱了。至
於日後托瓦茲會不會被蘋果族列為拒絕往來戶，還得看事件的後續發
展情形了。

■ 「PitBull LX」防禦攻擊解決方案將支援Itanium

沈美君/編譯
2001/04/17

以網路為交易平台的公司與日遽增，系統安全性亦不容小覷。 Argus
Systems  Group公司所研發的「PitBull LX」防禦攻擊解決方案，可
為個人或企業伺服器平台提供安全機制，目前已支援 Linux版本，並
預計推出支援英特爾Itanium企業伺服器的版本。

------

Argus Systems Group 公司致力於提供一系列的電子商務安全解決方
案與應用軟體，經過ITSEC 認證核可，支援多種平台：從桌上型電腦
到大型主機，可透過網路作調校設定。旗下研發的「PitBull LX」防
禦攻擊解決方案﹙intrusion prevention systems﹚， 已支援Linux
版本，預計在今年第三季，推出支援英特爾 Itanium企業伺服器的版
本﹙支援Linux核心2.4版﹚。

今年3月甫獲德國 ASP 集團組織頒發「最佳安全技術解決方案﹙Best
 Technical Security Solution﹚」獎項的 「PitBull LX」，被視
為容易安裝、具成本效益、高安全性的阻絕攻擊解決方案。企業伺服
器管理者，可透過PitBull LX，快速建置以網站為基礎、安全的電子
商務環境，可防止惡意攻擊，例如企圖竄改網站檔案、竊取私人檔案
、金融資訊、預防阻斷式攻擊﹙Denial of Service，簡稱DoS，企圖
在短時間內對特定網站送出大量要求，使網站癱瘓的惡意攻擊行為﹚
。

「PitBull LX」以作業系統層的安全機制，來實作其解決方案，獲得
國家安全局、Gartner Group、及許多產業專家的肯定，因「PitBull
LX」也支援Linux 平台，使企業可以用相當具有彈性的價格，建置高
安全性的作業環境。此外，「PitBull LX」宣布將在今年第三季推出
支援英特爾 Itanium企業伺服器的版本，無疑為企業客戶帶來更具效
能與競爭力的解決方案。

eWEEK雜誌也曾評選「PitBull LX 」為「最佳防禦攻擊／回應之產品
﹙Best Intrusion Defense and Response product﹚」。eWEEK在實
驗中證明了這個獎項的名符其實，在十六天中，「PitBull LX」承受
了五百四十萬次的攻擊，並且成功地作了防禦和回應。

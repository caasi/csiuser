■ Motorola進軍開放原始碼領域  投資Lineo兩千多萬美元

■ 英特爾開放電腦視覺資料庫  採BSD 授權

黃翔祺/編譯
2000/09/25

英特爾在日前舉行的IEEE Computer Society Conference電腦視覺及
特徵辨認會議上，宣布將開放其電腦視覺資料庫，該資料庫由C 語言
寫成，有上百個視覺功能，授權方式採BSD 授權，市場預料此舉動將
使得電腦視覺技術快速發展。

------

電腦視覺科技是利用演算法，抽離出移動或靜止的影像中有意義的資
訊，將以運算。在過去三十幾年，受限於運算速度的關係，要研究電
腦視覺只有在具有資金的大型實驗室中，使用超級電腦及特殊的設備
才做的到。

近年來，由於電腦設備的進步，在個人電腦上，就可以達到過去超級
電腦的運算效能，再加上影像擷取設備的進步，一般電腦技術人員可
以極低的價格，進行研究，這使得電腦視覺科技進步一日千里，相關
應用包括包括：手寫辨識、面貌辨識..到生物特徵辨識等。

現在，影像擷取資料隨著電腦及影像設備的進步而增加，但是相關的
技術卻沒有因此進步，原因是過去相關的影像演算法相當複雜，幾乎
是資訊「博士」程度知識才能做的到。為了加速電腦視覺科技的進步
，英特爾在IEEE的電腦視覺及特徵辨認會議上，宣布將開放其電腦視
覺資料庫，並提供Linux使用版本。

開放電腦視覺資料庫是由英特爾微處理部門所發展出來的，該部門除
了發展硬體架構之外，還針對電腦相關領域作廣泛的研究，包括：平
台架構﹙傳輸、記憶體﹚、及應用介面科技﹙包括自然語言、語音、
圖像等﹚。該資料庫中，包含著數百個功能的電腦視覺程式語言，從
擷取圖像校正、到圖形處理等，為了方便電腦技術人員研究與流通，
該程式資料庫是由C語言所寫成的。

目前，該電腦視覺資料庫原始碼的授權，是採BSD 授權模式，使用者
在修改影像資料庫時，只要提供相關版權說明，就可以進行該資料庫
的流通，而只要不涉及利用英特爾名稱作為商業用途，皆可以自由利
用該資料庫。

市場分析師表示在 英特爾 宣布提供開放電腦視覺資料庫之後，將節
省許多程式設計人員設計基礎的影像處理程式，專注於專門的影像處
理領域，這使得電腦視覺應用更加蓬勃發展。

■ 英特爾發表CDSA，普獲全球知名軟硬體廠商採用

沈美君/編譯
2000/09/26

英特爾Architecture實驗室主導研發的CDSA即將被遍佈全球的重量級
軟硬體廠商採用，以增加產品的安全性。採用這項機制的廠商除了有
康柏、惠普等硬體大廠，更有Linux套件的領導廠商如SuSE、Caldera
、TurboLinux等，CDSA將於開放原始碼執照將於十月底發佈，CDSA的
windows及linux版本也可在英特爾的網站上下載。

------

英特爾宣布，遍佈全球的重量級軟硬體廠商，例如康柏、惠普、SuSE
、Caldera、TurboLinux等，將採用英特爾 Architecture實驗室所主
導研發的CDSA﹙Common Data Security Architecture ，共同資料安
全架構﹚，增加產品的安全性。英特爾表示，將會在近期內釋出64位
元Linux版本的CDSA，以支援英特爾即將發表的Itanium高效能處理器
系列。

英特爾Architecture實驗室副總裁Craig Kinnie表示，廠商的協力發
展，以及開放原始碼的性質，使CDSA成為最能夠被發展者廣為接受的
安全機制。他指出，安全性對於產業的發展來講，仍是一個相當重要
的議題，而CDSA正可以幫助這些發展者透過這樣的安全架構，使開發
出來的產品更具安全性及競爭力。

CDSA由英特爾Architecture實驗室負責規格研發，適於合作性高、安
全性強的電子商務應用軟體。CDSA允許應用軟體透過編碼、生物學鑑
定、數位認證、授權認證等方式來取得資料權限。軟體廠商可以透過
CDSA架構，將這些安全機制加到自己研發的產品上，不用再自己產生
這些元件，將可縮短研發經費及時間。

康柏計畫將CDSA整合到Tru64 Unix作業系統平台上，而惠普也預計將
其整合至HP-UX。其他Linux套件的領導廠商也準備做機制的整合，如
：美國Caldera的OpenLinux、大陸紅旗Linux的Red Flag Server、德
國SuSE的Linux 7.1，與美國TurboLinux等。

英特爾表示，將會在近期內釋出64位元Linux版本的CDSA ，支援英特
爾 即將發表的Itanium高效能處理器系列。CDSA將於開放原始碼執照
將於十月底發佈，CDSA的windows及linux版本也可在英特爾的網站上
下載。

■ 「酷哥」，KDE上的商業報表程式

馮文龍/編譯
2000/09/26

「酷哥」﹙Kugar﹚ 是一套將商用軟體所產生的資料，加以套表加工
成高品質報表以供顯示或列印的應用程式。它支援Linux上的KDE視窗
環境，這使得KDE成為辦公室標準桌面環境的機會大增。

------

「酷哥」﹙Kugar﹚是以目前網際網路上非常熱門的XML為標準描述語
言，支援Linux 上的KDE視窗環境，這使得KDE成為辦公室標準桌面環
境的機會大增。

酷哥可以協助企業輕易地產生高品質的商業報表，酷哥可以將其他商
用軟體所產生的資料經由預先定義好的套表，快速地產生高品質的報
表，而這些資料及報表都符合XML的標準。

酷哥是由Kompany 公司所研發，這家公司的策略是整合多種不同的工
具，使得企業能夠容易使用並普及，並且整合這些不同的工具做所謂
第二階的銷售。Kompany的初期目標鎖定在專業的市場，像Powerlant

■ 開放原始碼的SSH軟體─OpenSSH 提供安全傳輸新選擇

謝良奇/編譯
2000/09/25

採用金鑰技術來辨識使用者身份的Secure Shell，能取代傳統的明文
式連線，提供安全的資料傳輸通道，是一種較為安全的通訊協定。今
年五月份釋出的OpenSSH 2.0，則是開放原始碼版本的SSH軟體，能補
足開放原始碼平台用戶的安全保障。

------

SSH ﹙Secure Shell﹚是一種新的通訊協定，具有傳統明文連線方式
所缺乏的高安全性。目前一般採用使用者名稱、密碼，來確認使用者
身份的明文連線，暴露的密碼容易受到駭客盜取，毫無保障的傳輸過
程，同樣威脅資料安全。採用金鑰對作為辨識身份的 SSH，可大大加
強連線過程的安全性，OpenSSH則是開放原始碼的SSH軟體。

在眾多封包解析程式氾濫，以及對資料保密、完整性需求節節升高的
今日，傳統網路連線採用的明文方式，已不符使用，Secure Shell即
是用以替代如telnet、rsh、rlogin 等明文式傳輸的新協定，除此之
外，更包括X11轉送、port轉送、鑑別轉送及Kerberos及AFS許可轉送
、IPv6等多項功能。

由Tatu Ylonen實作的SSH，因為授權的限制，使得Bjorn Gronvall決
定自行撰寫開放的SSH軟體，名為OSSH。此舉引起OpenBSD開發者們的
注意，在幾個月之內，釋出OpenSSH 1.2.2，並包含在OpenBSD 2.6之
中。OpenSSH 的授權十分自由，不論商業或非商業使用一律免費。

今年五月才剛釋出的OpenSSH 2.0，支援ssh 1.3、1.5及2.0，能自動
調整最佳協定，目前支援包括Linux、Solaris、HP/UX、Irix、AIX、
SCO及NeXT平台，接下來的開發，將著重於其它作業系統的移植。

使用者可以到OpenSSH的網站，下載該軟體的RPM檔案或其它支援格式
，安裝完成後，在正式使用OpenSSH 前，仍需要幾個步驟。首先必須
利用內含的ssh-keygen程式產生個人公開及私密金鑰，然後將公開金
鑰傳送到欲連線的遠端機器，透過OpenSSH的ssh，取代telnet等連線
程式，進行安全通訊。

■ Red Hat Linux 7.0甫一推出，戴爾等電腦大廠全力配合！

謝良奇/編譯
2000/09/26

功能經過大幅改進的 Red Hat Linux 7.0，一推出即引起轟動，包括
戴爾、Penguin Computing 皆在同一天宣布，將推出採用該新版作業
系統的產品，將進一步拓展Linux系統在企業領域的應用。

------

紅帽公司二十六日公開的 Red Hat Linux 7.0，挾著在安全、圖像等
功能的改進，甫一推出即獲得包括戴爾、 Penguin Computing的採用
。戴爾推出一系列搭載 Red Hat Linux 7.0的產品，包括伺服器、工
作站、桌上系統及筆記型電腦。Penguin Computing 宣布未來將在系
統上，向用戶提供Red Hat Linux 7.0。

具有產業領導地位的最新版 Red Hat Linux，能提供使用者更完善的
安全及管理功能，有助於推展Linux作為企業平台。

圖像及安全部份是 Red Hat Linux 7.0中最大的改革，採用2.2.16的
核心，包含XFree 4.0及名為Mesa的Open GL程式庫，能提高3D影像的
支援。由於RSA專利期滿，沒有專利權的束縛，Red Hat Linux 7.0的
安全能力也得以大幅提升，更內含Apache的SSH及SSL模組。

在Red Hat Linux 7.0 的開發期間，戴爾與紅帽早已密切合作，確保
該作業系統能平穩安裝在戴爾伺服器與客戶端產品，以及交互作業能
力。戴爾和紅帽合作的內容，主要是促進 Linux在商業上的應用，支
援企業建立網路基本架構。

戴爾推出的一系列產品包括PowerEdge伺服器、Precision工作站，將
採用 Red Hat Linux 7.0，桌上及筆記型電腦則以選擇裝置的方式，
滿足客戶的需求。

受Red Hat Linux 7.0吸引的不只戴爾公司，Linux網路服務系統供應
商Penguin Computing，亦宣布未來開始在系統上提供Red Hat Linux
7.0。

Penguin Computing總裁兼執行長Sam Ockman指出，該Linux系統最主
要的改進，在於網路服務及桌上計算方面，配合該公司高可信度的企
業級系統，與頂級的服務及支援，將能為客戶的企業網路增添新的動
力。目前搭載Red Hat 7.0的Penguin Computing系統，只提供少數特
殊用戶，預計將於十月中旬推出。

weinong是一位很有趣的同事，與他共事的人都這麼覺得。 他的名字
跟一位大學校長同名，但是本人完全不會給人嚴肅的感覺，微笑的時

，便不可自拔地迷上了寫程式。在網基，weinong 除了協助我們進行
OpenScope的新聞編輯工作外, 也支援撰寫OpenScope電子報部分的程
式。 

weinong 對工作非常地敬業，對於不熟悉的電腦語言，也都主動去研
究學習，並盡力解決程式寫作上的困難。然而就在不久前，他突然接
到兵單，隨著報到日一天天地接近，我們的心情愈加沉重與不捨，而
weinong 也捨不得網基，更加賣力地工作，甚至連報到的前一天晚上
，犧牲與家人和女友相聚的時間，只為了讓OpenScope 網站能順利地
如期上線；這樣認真努力的精神，真是太令人感動了，也因此讓我們
對他的離去感到萬分不捨。幸好weinong 已經答應留職停薪，相約未
來再為網基效力。 

自從九二一那天weinong 去從軍報國後，離現在已過了一個禮拜，不
知道他現在過得如何？當我在思考這個問題時，耳邊傳來一首 ─The
Cranberries的"Forever Yellow Skies" ，正能說明我現在的心情。
我們在網基的同事們都非常地想念他，也希望weinong 在經過成長的
粹鍊後，能夠趕緊回來，為了他所喜愛的程式設計與文件寫作繼續努
力。

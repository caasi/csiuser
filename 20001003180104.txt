■ 看！開放原始碼的未來電視

黃翔祺/編譯
2000/09/29

今年的國際傳播展除了在硬體上的進步之外，在軟體上也有幾項重要
的發展趨勢，第一是由於微軟欲全盤控制WebTV 軟體，使得多數廠商
決定選擇性能穩定的Linux作為機上盒﹙Set-top box﹚的作業系統。
其二是在開發軟體上，過去獨霸的跨平台開發軟體Java正遭受嚴厲的
挑戰，由開放原始碼社群發起的Kaffe運動，試圖挑戰Java的地位。

------

今年在國際傳播展上有幾件令人注目的事，這些在傳播上驚人的發明
包括有：巨大的平面電視、動態影像捕捉軟體....等等，而支持這些
驚人軟體的平台，正是最近大受注目的開放原始碼軟體。

以Nokia所推出的Nokia Media Terminal 為例，除了支援寬頻功能之
外，透過小的鍵盤，更可以上網、收發email 、如瀏覽器般，設定自
己最喜歡的頻道、並且具有隨選視訊的功能，看起來就跟個人電腦沒
什麼兩樣，而這終端機所使用的軟體正是Linux、XFree86、 Mozilla
及部分Nokia所發展的軟體系統。

那麼微軟過去所發展的 WebTV系統呢？多數廠商表示由於微軟試圖獨
佔整個WebTV軟體系統，引起廠商的不滿。再加上Linux系統穩定的特
性，多數的廠商放棄WebTV轉而投向Linux的懷抱，消費者表示不能夠
忍受過去在電腦常發生當機狀況，分析師表示誰願意在看世界杯足球
賽之中，發生當機的狀況。

在這次展覽中，除了軟體系統產生重大改變之外。在開發軟體上，長
久以來被大家讚賞的跨平台開發軟體Java，因為昇揚編譯器執行速度
緩慢，再加上昇揚改進核心程式的緩慢，開放原始碼的組織決定發展
 Kaffe來取代Java，Kaffe 未來的發展成為本次展覽另一個注目的焦
點。

■ 支援多螢幕 Linux用途更寬廣

張書銘/編譯
2000/09/29

在人來人往的街上，電視牆絢麗繽紛的畫面是否常令人為之駐足。但
在欣它的同時，大家往往想不到，究竟在後端操控著這多螢幕展示環
境的到底是怎樣的系統？現在不必再懷疑了，隨著Jupiter System公
司的發展，掌控多螢幕，Linux也可以！

------

多螢幕顯示在視窗系統上很早前就可以做到，但是 Linux上並沒有特
別針對這方面的功能多做發展。這和 Linux原本對畫面並沒有特殊需
求的性質有關。但是隨著多媒體能力需求大增，在 Linux上也不得不
想辦法來彌補它的不足之處。因此，以提供網路基礎多螢幕顯示設備
與方案而著名的Jupiter System正式宣布將支援Linux。

一九八二年就成立的Jupiter System公司原本是以設計螢幕為主。後
來開始轉型。除了螢幕硬體本身以外，也研發相關的人機互動設備，
針對以UNIX為主的平台提供軟體以提昇顯示器的效能，像Open VMS、
Solaris等都是Jupiter System的支援對象。也因此 Jupiter System
的產品無論在個人電腦或電視牆市場上都有高知名度。

而這次Jupiter System宣佈對Linux提供支援的是Jupiter 870系列顯
示牆控制器。這系列一共有兩種款式，分別是 Jupiter 870配備單顆
Pentium III 600處理器以及配備了兩顆 Pentium III 750 處理器的
Jupiter 870mp，而這兩款都可支援最高到768MB的ECC記憶體。

Jupiter  870系列提供了最高共124 台顯示器的控管功能，並且提供
Live Video/analog RGB 畫面支援。而且可以掌控最高達十組乙太網
路的資源，以同時處理大量資訊流。因此在Jupiter System初步的規
劃上，這兩款支援 Linux的控制器主要將應用在大眾運輸業、交通控
制以及金融業如股市看盤系統上。

■ 紅帽新網站 採用Novell新技術

葉志博/編譯
2000/09/28

最近才開張的紅帽網站，基於為眾多的Linux 使用者提供更加安全的
網路服務環境，採用了Novell的eDirectory和DirXML技術；另一方面
，Novell近日在記者會上發佈了一連串利多消息，企圖轉移市場分析
師對Novell不利的言論或臆測。

------

紅帽這陣子動作不斷，前幾天才推出最新的紅帽 Linux 7.0版，同時
介紹了紅帽Linux網路，現在又宣佈採用Novell公司的 eDirectory，
為眾多的紅帽網站使用者提供目錄服務。紅帽網路主要的功能是為全
球使用Linux 的企業或程式設計師提供穩固的線上支援，基於業務和
穩定性的考量，以及為了和現有的系統能夠緊密地整合，促使紅帽選
擇使用Novell的eDirectory和DirXML技術。

網際網路是發展、建構和管理作業平台的未來趨勢，而eDirectory不
但增加了Linux 系統的安全性和穩定性，更可減少系統管理的成本，
確保紅帽網路為每一客戶提供量身訂作的專業服務，使分散的系統能
夠以更為有效率且集中化的方式來管理。

而DirXML功能的目的是用來加速轉變傳統企業成為電子商務，它擴充
了eDirectory的功能，使得企業能夠在第一次就能完全地將營運程序
和現有應用程式完全地整合。由於採用eDirectory和DirXML，讓紅帽
在發展階段上，節省了大量的時間及金錢。

此次合作案對Novell而言，紅帽不但是客戶也是夥伴，但也是Novell
用來沖淡客戶和投資大眾疑慮的籌碼之一。經過了激烈的競爭，下滑
的財務報告，以及市場上對公司未來的揣測，Novell，這個網路軟體
市場上的老牌供應商，也不得不採取些行動來證明它的實力，並藉此
增加大眾對它的信心。

在產品方面，Novell 將推出eBusiness網路入口服務；在客戶方面，
除了essentialtalk公司採用Net網路服務軟體，英國電話公司也選擇
Novell提供企業目錄服務，並簽訂了Netware更新合約。Novell 也推
出了新的ICS﹙Internet Caching System﹚1.3 版，提供安全的站台
存取，高流量站台的管理，以及網站過濾等功能。希望這些有關新合
夥關係、技術發展的發表，能夠使財務分析師的評估趨於緩和，最重
要的還是廣大的客戶及投資人能夠對Novell恢復信心。

■ Linux上的Visual Basic誕生了！

謝良奇/編譯
2000/09/29

對Windows的開發者而言，轉移到Linux平台進行開發工作，現在又多
了一項有力誘因，尤其是針對微軟的Visual Basic程式設計師，因為
Linux上出現一套採用VB般視覺介面，並且完全支援物件導向的Basic
開發工具，名為Phoenix Object Basic。

------

Janus Software公開最新版的Phoenix Object Basic，這是一套獨特
的Linux快速應用程式開發﹙RAD﹚工具，結合Visual Basic的易學易
用，及Python、Perl等物件導向程式語言的表達能力，著重於縮短陡
峭的學習曲線，為欲轉移到Linux平台的Windows程式設計師，提供物
件導向式的發展環境。

Phoenix Object Basic是一套Linux的物件導向式RAD開發工具，完全
的物件導向支援，使得該開發工具具備如Python、Perl等多種程式語
言的強大功能。該套工具採用如Visual Basic般的視覺設計介面，易
於學習使用的特色，有助縮短陡峭的學習曲線，十分適合已有VB設計
經驗的Windows應用程式開發者，在轉移到Linux平台時，仍能應用過
去累積的基礎。

最近的市場研究顯示，全世界約有七百五十萬個Visual Basic程式設
計師，其中有近百分之四十的比例，今年將轉移到 Linux系統上進行
開發。此時一套適合Visual Basic程式師使用的工具，就變得格外重
要。Janus Software推出的此套開發工具，將能吸引更多開發者加入
Linux行列，參與開放原始碼運動。

但在授權方面，雖然Phoenix Object Basic可以自由下載，卻並非開
放原始碼，而採用共享軟體授權。根據Janus Software的說明，儘管
該公司深信開放原始碼的優點，但在Phoenix Object Basic採部份原
始碼由第三方授權使用的情況下，使得該公司必須以封閉原始碼發表
Phoenix Object Basic。

Janus Software宣稱，目前正在努力移除由第三方提供的元件，預計
將在明年第一季時，公開Phoenix Object Basic的原始程式碼。此外
，該公司也將持續在該發展工具對資料庫、網路、XML及GUI的支援，
進行一系列改善。

■ 「自由」瀏覽器 讓你上網安全無拘束

張書銘/編譯
2000/09/29

近年來隨著開放原始碼軟體快速的發展，各類應用程式也在Linux 上
蓬勃發展，使得Linux 在網際網路上獲得好評。而網際網路上最風行
的應用─WEB ，當然也有許多相關應用程式將其原始碼釋出，讓大家
更容易接觸並習慣使用它們。

------

過去在瀏覽器市場上最著名的兩套產品就是微軟 Internet Explorer
以及網景通訊家族。在當時因為微軟將IE與視窗作業系統“整合”在
一起；挾視窗系統市場佔有率來和網景競爭，確實讓網景吃了不少苦
頭。也因此網景投身開放原始碼環境藉以反擊，讓微軟體認到這股力
量的可怕，也讓不有少開發瀏覽器的個人以及公司有了新的觀念，決
定開放原始碼來增加自己產品的競爭力。

Zero Knowledge就是其中一個例子。Zero Knowledge公司在一九九七
年成立，當時全世界的網路風正開始延燒，因此Zero Knowledge致力
於開發保護網路使用者資料的各種應用工具。Zero Knowledge並於九
九年年底時推出了 Freedom這套可以提供使用者完整保護的瀏覽器以
及相關的工具組，而 Freedom完整的網路安全機制也在市場上造成了
相當大的回響。

為了讓這套以安全性著稱的產品能更為市場所熟悉，Zero Knowledge
決定在 Linux上推出此產品，同時開放它的原始碼。Zero Knowledge
表示做出這項決定是同時考量到使用者以及其他開發者兩方面，Zero
Knowledge希望使用者能在接觸到Freedom原始碼後能了解到它驚人的
安全防護能力，讓使用者能安心選用 Freedom；而針對其他開發者則
是希望能提昇大家的網路安全能力，讓網路上的各種操作更為安全，
同時也希望能因此找到自己的破綻並加以改進。

目前Zero Knowledge宣傳Freedom可以提供SSL安全機制，並對電子郵
件、網路聊天﹙IRC﹚、新聞群組以及 telnet等各種服務提供安全控
管，避免目前各種可能的網路追蹤技術以保障使用者。這套 Freedom
將在今年的十一月間在linux上推出並釋出原始碼。

■ 藍鳥計劃易手  Atipa收購領導OpenNMS.org公司

蕭喜那/編譯
2000/09/29

Atipa宣布收購領導OpenNMS.org計劃的公司，名稱為 PlatformWorks
，由OpenNMS計劃創始人所成立，OpenNMS從事下一代網路和系統管理
軟體開發。此項收購將大大提升Atipa在開放原始碼市場的地位。

------

Atipa公司近日宣布收購創立OpenNMS.org的公司，OpenNMS.org 是一
項開放原始碼計劃，從事下一代網路和系統管理軟體的開發，此項軟
體預計將於明年年初發表。OpenNMS的管理小組將加入Atipa，並領導
Atipa 旗下的子公司，這家子公司將提供完整的專業服務和支援，給
應用OpenNMS軟體的企業。Atipa此次的收購將使該公司擁有全新和獨
特的企業科技，這項科技可說是網路管理的革命，可用於掌握蓬勃的
全新商機。

Atipa是終端對終端Linux和開放原始碼解決方案的主要供應商，它也
將利用這項收購行動，以擴大該公司的開放原始碼產品和服務市場線
、企業顧客基部、以及它與開放原始碼社群的關係。此收購使 Atipa
的開放原始碼企業解決方案家族，多了世界級的網路管理服務，同時
也提升了Atipa的形象，成為具領導性的開放原始碼廠商。

Atipa收購的公司現在的名稱為Platform Works，最早是由OpenNMS的
創辦人所成立，OpenNMS 已經根據GNU 公共授權﹙GPL ﹚的規定，將
名為Bluebird的下一代軟體授權出去，GPL 是現行的開放原始碼授權
辦法，它可以確保開放原始碼軟體的繼續開發，同時避免有人出於專
有的目的，而申請擁有該軟體的專利。

Atipa的董事長兼執行長Jeffrey Keenan 對這項收購發表看法，他認
為OpenNMS 的開發平台可以製造出一項開放原始碼的產品，而這項產
品最終將改變企業管理市場的面貌，不像今日逐漸老化的科技，Open
NMS 是經過特別設計的企業解決方案，以符合網際網路路經濟對網路
和管理的全面性需求。一旦OpenNMS取得市場佔有率，已經加入Atipa
的 OpenNMS科技領導人和發明人，將使Atipa的支援服務成為業界的
標準。

■ Abriasoft與Linuxsolve合作 讓你的MySQL更安全！

蕭喜那/編譯
2000/09/29

Abriasoft 為致力於開放原始碼資料庫社群的資料庫公司，日前宣布
將與網際網路安全伺服器裝置廠商LinuxSolve合作，組成策略聯盟，
以期推出安全的mySQL 網路裝置，縱橫安全伺服器和線上資料庫市場
。

------

Abriasoft 是位於矽谷的一家資料庫解決方案的供應商，近日宣布與
Linux-Solve 組成策略聯盟，LinuxSolve在開發網際網路安全伺服器
裝置上，具有領導性的地位，也是第一家銷售可以在Linux 作業系統
上運作的安全伺服器裝置的公司。

Abriasoft 目前正致力取得開放原始碼資料庫市場的領導地位，因此
，它積極與LinuxSolve組成策略聯盟。這次Abriasoft 與LinuxSolve
的合作，結合了硬體和軟體兩方面的資源及技術經驗，可以提供顧客
更多、更完整的電子企業解決方案。兩家公司策略聯盟包括技術開發
計劃和共同的市場行銷活動，以期在安全伺服器和線上資料庫市場上
更具影響力。

Abriasoft的執行長Paul Afshar對這項合作發表他的看法，他認為安
全問題對身為線上資料庫供應商的Abriasoft 而言，是需要優先考慮
的，同時，他指出LinuxSolve的安全伺服器裝置對該公司MySQL 的未
來，具有相當的重要性。

LinuxSolve的執行長Thmoas Knapp也對這項策略聯盟發表看法，他認
為，與Abriasoft 的合作，對於「動態全球資訊網內容出版」伺服器
裝置市場上的顧客而言，將是一大利多，而這些伺服器裝置具有高階
安全性、穩定性、容易與LinuxSolve的GuardWorks安全伺服器裝置作
業軟體搭配使用，價格又相當合理。他認為兩家公司在產品開發上的
合作，將可使MySQL 被更多人採用，以作為全球資訊網基礎建設的後
端資料庫解決方案。

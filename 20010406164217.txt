■ Linux今年成長率可望超過150％

李孟諺/編譯
2001/04/04

由freeVSD計劃贊助，Idaya公司進行的調查顯示，Linux 今年的市場
成長率預期將超過150％。並可望在 2002年年中成為網站伺服器平台
作業系統的主流。而且，開放原始碼軟體也有信心在2003年成為網站
管理技術的領導者。此外，VA Linux公司將獲利日期延後到2002年的
10月。

------

Linux成長速度驚人，由 freeVSD計劃與Idaya公司共同主持的一項研
究調查，結果顯示，Linux今年的市場成長率預期將超過150％。這項
調查時間是從今年1月到3月，範圍涵蓋全球前一千大的 ISP當中，已
經在 GPL通用公共授權與應用投入大量經費的公司。回覆調查者以混
合商務管理階層、科技領導者，與服務管理者為主。

促成Linux成長的原因有功能上的改善，特別是Linux應用能適用在許
多標準產品，以及故障安全防護，如檔案系統紀錄功能等。但是潛在
妨礙Linux 市場成長的因素，有相關套件與應用開發者支援能力的不
足、市場的保守以及對採取GPL授權解決方案的不信任。

研究調查的結果：以去年成長的速度為基準，Linux 市場在今年可望
成長154％，並預計在 2002年中可以主宰網站伺服器平台。尚有45％
的受訪者期盼開放原始碼軟體能於2003年時，能在網站管理技術方面
稱霸，但是有33％受訪者則不認同。至於要如何達成這個目標？66％
的受訪者認為軟體功能必須改善，53％認為檔案系統紀錄功能的進步
，有48％的人則認為是安裝便利性的增加以及改善 GUI圖形使用者介
面，能加速成長速度。

但是有將近三分之二的ISP 受訪者表示，相較於微軟之類的軟體，開
放原始碼軟體必須達到企業等級的應用需求，市場的懷疑與支援的可
獲得性成為妨礙 Linux市場成長的潛在因素。此外，有70％的受訪者
認為Linux應該與freeBSD整合，以便於 Linux套件能結合他們的功能
與開發工具在標準的Unix平台使用。

根據此項調查，Linux 未來的市場商機無限。但是在整體經濟環境不
景氣的情況下，仍有許多Linux公司延後獲利日期。提供Linux作業系
統軟體與產品的VA Linux公司，宣佈裁減該公司25％的員工以及將獲
利日期延緩到2002年10月，比前原先的預期晚了九個月。VA Linux執
行長Larry Augustin仍表示，該公司的獲利狀況都如預期般的成長。
但該公司的股價卻已經從最高的一百零三美元跌至4月3日的收盤價只
剩約二點零九美元。

■ IBM新思維  Linux將徹底改變資訊產業

梁天佐/編譯
2001/04/04

在電腦資訊產業的頭號鐵律─「不要和你的競爭者分享你的秘密」之
下，專利軟硬體切割、分化市場，行銷廠商藉此控制開發人員與使用
者，產業與軟硬體的發展於是偏離供需的常軌，造成壟斷；曾經以此
鐵律稱雄業界的藍色巨人IBM ，最近頻頻以大動作改弦易轍，以新思
維擁抱Linux。

------

IBM技術與策略部門副總裁Irving  Wladawsky-Berger在IBM官方網站
發表的一篇文章中表示，由於實際的需要與潮流， Linux具有的特性
將從根本改變資訊科技產業，IBM因為看到這個發展趨勢，乃對Linux
大舉投入人力與資金， Wladawsky-Berger說：「IBM從擁有最多專利
的公司，變成這項資訊科技新嘗試的冠軍。」、「我們看到曙光。」

文章中指出，IBM將在Linux相關分野上投資一億美元，同時投入一千
五百名程式設計師研發，讓 IBM的每一項軟、硬體產品都能相容或操
作Linux，以加速 Linux被採用做為企業平台；IBM認為，提供最具彈
性給客戶的作業系統將贏得最後的勝利，Wladawsky-Berger在文章最
後說：「我們把IBM未來的一大部分，押注在Linux上。」

如果Linux 只是一款在商業市場上競爭的作業系統，它所受到的注意
與期待不會如此熱烈。開放原始碼的特色，除了使其在修正錯誤、增
進功能的機制靈活且有效，最重要的是提醒世人軟體被製造與遞交給
使用者應有與合理的方式，所以終將形成一股潮流。

崇高的理想之外， Linux還有兩項重要的特性足以改變資訊科技產業
。一是Linux 幾乎可以在任何硬體平台上執行，意味著企業或個人只
需做極小的更動，就能很快利用Linux 的好處；另外一個則是它改變
了廠商對客戶的價值，廠商必須提供客戶真正需要的服務，而不是以
控制來逼迫客戶就範。

雖然部份開放原始碼社群或個人對IBM大動作投入Linux發展表示憂心
，認為IBM會藉此挾持Linux ，將Linux的成果據為私有，但就目前發
展的狀況與實際的行動看來，IBM確實以新的思維與做法對待Linux。
IDC系統軟體研究部門副總裁Dan Kusnetzky便說 ：「IBM比其他廠商
展現更強的Linux投入政策，是因為大家對Linux 很興奮，IBM要使所
有的產品反射它的光芒。」

■ GNOME 1.4版釋出   新增Nautilus網頁瀏覽器

沈美君/編譯
2001/04/04

有段時間未推出新作的GNOME，於2日發表 GNOME 1.4版，加入新的圖
形介面檔案管理員Nautilus，和眾多新特色。不過稍後因為意外發現
的臭蟲，將發表時間延後數小時，並緊急進行修正。

------

GNOME基金會﹙GNOME Foundation﹚於2日發表最新的桌面整合環境「
GNOME」1.4版，但稍後因為GNOME中 有意外發現的臭蟲，發表時間比
預期稍晚，並緊急進行修正。

據稱，GNOME 1.4 除了較上一個版本穩定之外，並加入新的圖形介面
檔案、網頁瀏覽器「Nautilus 」﹙原由Eazel公司所發展﹚，以及許
多新版或修正過的應用軟體及工具，並緊密地整合線上支援文件，使
GNOME成為更具親和力的桌面環境，更理想的程式發展環境。

導致發表時間延遲的是「Nautilus」的關鍵部分「Medusa」，Medusa
負責硬碟檔案資料的編目工作﹙indexing﹚，可加速使用者對資料的
搜尋動作。GNOME發展經理人Jacob Berkman指出，Medusa可能會導致
處理程序中的file descriptors漏失﹙file descriptors─ process
開啟檔案所產生的 file handle﹚。

GNOME 1.4 對一般桌面使用者及程式發展者，都作了相當的加強。在
桌面使用者方面，新的檔案管理員「Nautilus」同時可作為網頁瀏覽
器使用，「Sawfish」也加入成為GNOME視窗管理員一族。新版 GNOME
讓線上求助文件以更具親和力的方式呈現，並且增強與KDE和其他X視
窗應用程式的整合。對程式發展者而言，GNOME 1.4 的加強是一大福
音：更新Bonobo Component Model﹙提供標準的程式寫作元件，加強
Linux與UNIX的相容性﹚、提供 Xml-i18n-tools﹙作語言層面國際化
與本地化的工具﹚、支援GNOME-VFS﹙虛擬檔案系統﹚。

目前，全球約有數百萬的使用者以GNOME 作為桌面整合環境，並有超
過五百位程式設計師致力發展 GNOME，其中有一百多位全職受薪的發
展者。

■ Qt大方送  吸引年輕學子加入開發

李孟諺/編譯
2001/04/05

中美戰機空中交鋒，誰能勝出尚未知曉。兩大Linux桌面環境系統KDE
與GNOME的戰役也互有領先，鹿死誰手亦難預測。但是，KDE的基礎架
構Qt決定前進校園向下扎根，培養新的生力軍。 GNOME未來似乎還有
一場硬仗要打。

------

開發Qt、C++跨平台工具以及視窗系統的Trolltech公司，為了吸引更
多使用族群，決定抓住年輕學子。藉由釋出Qt給在大學電腦科學相關
科系的學生，該公司希望能為Qt帶來一群新的開發者。一旦此計劃成
功，另一個Linux桌面環境GNOME將有一場硬仗要打。

Trolltech執行長Haavard Nord 表示，全球許多電腦科學相關科系都
已採用在Unix開放原始碼版本上使用的Qt。透過該公司的教育學程計
劃，電腦相關科系將可獲得免費的視窗Qt授權。該措施將會減低相關
科系的經費負擔，亦能讓教授們能知道Qt的跨平台功能，是非常適合
學生做為進入將來涵蓋多種作業平台的市場技能。

一般說來，Qt有三件事比技術還重要，一是跨足到主流學校，其次是
是用在嵌入式系統，最後是輕便性。從第一點來說，以往電腦相關科
系只能免費試用在Linux版本的Qt，透過Trolltech的Qt大學學程計劃
，視窗版本的Qt也將可免費取得。這是個雙贏的策略。大學可以免費
獲得視窗平台的Qt，而Trolltech 則會獲得許多使用Qt的開發者。

此外，專門設計給嵌入式設備使用的Qt/Embedded，不像X視窗需要耗
費許多記憶體空間。 Qt/Embedded利用frame buffer技術，使得它能
夠在PDA 或手機上創造出小而快的應用，符合未來資訊家電用品體積
小功能多的潮流。

這就是Qt在未來比GTK與GNOME佔優勢的地方。當然，其對手Ximian與
Eazel公司未來的計劃還不明確，而且昇陽公司也背書將採用GNOME為
OpenOffice計劃的桌面環境。不過他們並沒有將GTK 推展到校園當中
。即使做到這一點，在支援視窗與嵌入式系統的功能上，Qt還是較佔
優勢，GNOME仍然有硬仗要打。

■ 提昇對Linux相容性  HP開放多項程式原始碼

張書銘/編譯
2001/04/04

由於開放原始碼風氣日盛，許多公司紛紛釋出特定產品的原始碼，以
加強產品與各種系統間的相容能力。惠普公司也不例外，過去幾個月
來，惠普公司提出了不少關於開放原始碼的計劃，並致力提昇公司產
品與Linux的相容能力。

------

在大眾的期盼之下，惠普公司終於開始針對 Linux釋出多套程式的原
始碼。除了印表機的驅動程式之外，還包括了用於新款印表機的列印
伺服器軟體 Print Server Appliance 4200。據惠普公司資料顯示，
除了一些設備驅動程式與伺服器應用程式外，未來數年間惠普還將撥
出預算，專門從事 Linux的相關開發。

過去惠普公司雖然提出過開放原始碼的規劃，但遲遲沒有行動，因此
在三月下旬正式宣佈這些開放原始碼的訊息後，受到開放原始碼社群
的大力讚賞。過去幾週內，惠普公司連續釋出了多套不同程式的原始
碼，包括了用於惠普 Deskjet 600至Deskjet 990C之間型號的印表機
驅動程式以及Print Server Appliance 4200 Linux 版等，這些軟體
都將以GPL或LGPL授權釋出。

在這些程式中，最受到注目的是 Print Server Appliance 4200，HP
 Print Server Appliance 4200是一套由惠普以及Samba開發團隊共
同研發的列印伺服器軟體，提供IT人員網路管理、安裝、設定列印伺
服器組態等功能，可支援使用微軟 Windows95、98、NT、2000及UNIX
等系統的客戶端。搭配上惠普LaserJet系列的引表機，可對來自網際
網路或無線通訊裝置的資料做更完善的列印處理。

除了針對 Linux連續釋出了多項程式原始碼外，惠普公司也成立了專
屬的 Linux社群，同時來自惠普的資料顯示，公司將編列預算，專門
開發Linux的相關產品，因此Linux的使用者在未來幾年內必定可獲得
來自惠普公司更完善的列印系統支援。

■ PostgreSQL推出Solaris平台測試版

沈美君/編譯
2001/04/05

使用Solaris系統平台的企業，又多了一項資料庫的新選擇。由Great
 Bridge公司推出，適用於Solaris平台版本的PostgreSQL 商業測試
版及相關技術支援，將開放原始碼的旗艦級資料庫，推向電子商務企
業的懷抱。

------

Great Bridge公司最近動作頻頻，日前才宣布與 Zend Technologies
公司合作，將推出結合 PHP與PostgreSQL的動態、資料庫導向網站開
發套件，接著又於3日宣布推出適用於Solaris平台版本的PostgreSQL
商業測試版及相關技術支援，稱為「PgSol  project」套件計畫，將
開放原始碼的旗艦級資料庫， 推向電子商務企業的懷抱。

PostgreSQL為物件關聯式資料庫，是開放原始碼資料庫中，功能最完
備、並足以媲美商業等級的資料庫產品；而昇陽 Solaris是企業界最
為廣泛使用的伺服器平台。Great Bridge發表Solaris版 PostgreSQL
的舉動，將開發許多潛在的企業用戶。Great Bridge預期在今年中前
提供全套的PostgreSQL商業支援，「PgSol」套件是第一步。

過去，使用Solaris 系統平台的企業，幾乎沒有其他選擇，只能付大
筆費用，從Oracle等公司購買商業資料庫的使用權。Great Bridge最
大的競爭優勢，就是降低的軟體購買成本。Great Bridge總裁Robert
 Gilbert表示，PostgreSQL可以滿足Solaris社群的資料庫需求。

PgSol套件包含完整的 PostgreSQL資料庫、相關工具、管理操作介面
及文件，Great Bridge將眾多相關檔案，壓縮成單一套件，使用者可
以一次完成下載動作。管理者可使用Solaris的套件管理員﹙Solaris
 Package  Manager﹚快速安裝此套件，不用再耗時費力地從原始碼
來編譯安裝。Great Bridge提供專業的諮詢服務，為企業客戶在資料
庫建置、整合、相關軟體設計、套裝應用程式開發或嵌入式解決方案
，提供專業技術支援。

成立剛滿一年的Great Bridge，主要提供以PostgreSQL為基礎的開放
原始碼解決方案，目的在以極低的成本，提供企業級的資料庫與相關
工具，因為是開放原始碼，企業用戶不用再受制於封閉的廠商解決方
案。

■ OnCore實現嵌入式系統共和國夢想

張書銘/編譯
2001/04/04

過去想要順利執行為不同系統撰寫的程式，除了等待移植版本外別無
他法。現在嵌入式設備要突破這個限制，利用混合作業系統環境的想
法讓不同環境下的應用程式能順利執行。即時作業系統供應商OnCore
公司日前宣佈為 VxWorks作業系統研發出了新的解決方案，從此混用
多種系統下的應用程式再也不是夢。

------

OnCore公司正式宣佈為知名嵌入式設備作業系統 VxWorks開發出新的
解決方案，讓 VxWorks下的應用程式無須經過繁雜的移植手續，也能
在其他作業系統如Linux、pSOS、POSIX下運作。這項方案的推出，不
但使程式開發者負擔減輕，對需求複雜的使用者更是一大福音。

VxWorks系統是目前最大的嵌入式軟體公司 Wind River的核心技術，
而由於 Linux在嵌入式設備上的佔有率逐步提高，因此自去年起，該
公司便提供開發工具VxWorks-to-Linux，讓程式設計者把為 VxWorks
設計的嵌入式應用軟體，轉為嵌入式 Linux應用軟體。這套工具是以
虛擬機器的方式來執行應用軟體，以一個專門用於轉換的函式庫，將
VxWorks的API轉為與 Linux相容的函式。但由於使用過程繁複，開發
者仍要花許多時間才能順利將程式轉換。

OnCore公司為了解決這樣的問題，採用了混合作業系統環境的方式。
OnCore公司的做法是，利用虛擬記憶體空間置放各系統下的應用程式
，並虛擬其他系統的核心，讓應用程式認為目前的作業系統是可以執
行的環境，這樣的做法可以完全不必重新撰寫程式，只要安裝好虛擬
系統的軟體即可。

根據OnCore公司表示，這項解決方案已接近完工，將於今年多項展覽
中正式展出，年中前已確定的參展行程包括：4月10-11日於舊金山的
嵌入式設備展；5月8-10日，拉斯維加斯的 NetWorld與InterOp Show
；5月16-17日倫敦的2001年ESS；和6月5-7日亞特蘭大SuperComm。

■ Linux在高等教育扮演的角色

梁天佐/編譯
2001/04/05

開放原始碼的精神其實非常簡單明白，開放的胸襟與做法要求的是精
確、進步、公平與正義；高等教育如大專院校不論是在學術研究需要
或社會風氣的引導上，採用Linux 之類的開放原始碼軟體，不僅能具
經濟效益，研究與教學成果更有公信力與成效，還能促進社會公平。

------

在高等教育機構內提供非資訊科系學生基礎資訊科技教育，如作業系
統、辦公室應用軟體與網際網路能力有其必要，但是建置與維護數量
與規模不算小的個人電腦、伺服主機與網路架構所費不貲，再加上採
購、升級與軟體授權的花費，確實是學校、政府與社會的沉重負擔，
因此在經濟效益上，Linux與開放原始碼軟體便具有極高的適合性。

雖然Linux之類的開放原始碼軟體感覺起來不像Windows或Office那麼
方便，但由於學校教育的重點在於教導學生對資訊科技的掌握與了解
，Linux 手動設定的彈性、支援公共標準與既有系統相容的特色，正
好提供教學需求；在學術研究上，應用Linux 與開放原始碼應用軟體
進行研究，不僅穩定，效能符合需求，最重要的是研究工具、過程與
成果可以被公開檢證，具商業封閉原始碼軟體所無法提供的公信力。

除了實務上的需要，在教育機構採用Linux 與開放原始碼軟體亦有其
社會公平與道德上的考量；調查報告指出，過去三十五年以來，人類
社會中「數位區隔」﹙Digital Divide﹚隨種族、貧富與地區等能否
取得資訊科技的差異加速擴大，因此更加深化社會財富、知識等差距
，偏遠貧窮地區要縮短數位區隔，開放原始碼才能提供經濟與知識上
的解決；墨西哥政府以Linux與GNOME為學校建置系統的原因即在此。
因此，高等教育機構有責任建立典範與引領潮流。

商業公司雖在校園推行「學生版」軟體，以低價提供師生使用商業軟
體，但這些只是商業公司壟斷市場，驅逐其他競爭軟體的商業手段之
一，因為在高等教育機構取得佔有率之後，可藉延伸的普及率操縱標
準，使得包括開放原始碼軟體在內的其他軟體相對不便或顯得昂貴，
配合以財力為後盾的立法遊說，進而消滅對手，獨占市場。

開放原始碼軟體公平、公開與具有實際效益的特性是社會正義的關鍵
，高等教育機構一旦採用開放原始碼軟體，研究發展出來的成果就能
被社會人群直接利用，以前這些是商業封閉原始碼軟體的生財工具，
只有特定社會階層的人才享受得到。

具有教育、學術、標準與社會典範功能的高等教育機構，不論就何種
立場，都應採用該放原始碼軟體為主流，不只在地區內，國際間也應
建立起相同的共識與標準，這在遠見國家與有識之士早已推動多時。

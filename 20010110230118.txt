■ Nokia開放原始碼資訊家電 年底美國上市

張書銘/編譯
2001/01/08

說到Nokia，多數人都會想到他們所推出的手機產品，但是現在Nokia
將帶來截然不同的產品─網路數位電視。這項利用開放原始碼平台研
發的產品，即將進駐美國家庭，讓網路家電深入日常生活當中。

------

去年9月Nokia推出以Linux為平台的Media Terminal，使用XFree86視
窗系統以及Mozilla 瀏覽器，打造出一套完全以開放原始碼系統架構
成的網路暨多媒體產品。配合 CABLE或xDSL技術，不但可提供電視功
能，也可用於上網。根據 Nokia評估，在今年年底Media Terminal將
可在美國全面上市。

Media Terminal的硬體方面採用Intel x86架構，擁有硬碟和PCI網路
介面，並加入 Nokia DVB-S接收器模組，透過這個模組提供完整的網
路存取能力，以及數位電視功能。同時在美國上市的Media Terminal
將會使用全新的Nokia NaviBars瀏覽器做為主要介面，提供更快速平
順的網路以及電視訊號傳輸。

Media Terminal是諾基亞針對「網路家庭」計劃推出的第一項產品。
目標是讓網際網路深入每個家庭。將網際網路媒體與數位廣播科技結
合，讓使用者能以單一顯示裝置來存取網路並接收電視媒體訊息。在
Media Terminal幫助下，使用者可以享受在家購物、網路銀行、電子
郵件與聊天室、播放 MP3、電視與網路同步子母畫面、數位電視錄影
與3D與互動遊戲服務，甚至可以暫停與重播現場節目。

Nokia表示，該公司在 IP技術、行動通訊與數位廣播方面的科技背景
是完成Media Terminal的重點之一，而基於開發過程中取之於社群；
用之於社群的理念，諾基亞將盡可能釋出相關技術，以刺激社群開發
Media Terminal應用程式的興趣。

■ PhatBOX進駐車用音響 讓MP3陪您走天下

張書銘/編譯
2001/01/08

MP3 是目前音質與檔案大小比最佳的格式之一，但受限於硬體技術，
要欣賞到最好的效果通常必須待在待在PC前。PhatNoise 公司為了打
破限制，選擇結合車用音響，以達到MP3帶著走的目標。

------

儘管 MP3目前仍牽扯到許多版權問題，但無庸置疑仍是目前最優秀的
數位音效檔格式。但MP3 檔案通常只能待在家中使用電腦來播放，想
利用 MP3隨身聽隨時欣賞，卻又受限於硬體技術的能力，不是可攜帶
的檔案太少，就是待機時間太短，無法盡興。而 PhatBOX結合車用音
響的構造，為使用者一次解決這兩樣問題。

PhatBOX在儲存裝置的容量上，比一般MP3隨身聽大了許多。它使用了
稱為PhatCart的儲存裝置，PhatCart 是可攜式、容量5到20GB、支援
ATA 技術的硬碟，和一般筆記型電腦用的硬碟大小相近。若以一般音
樂CD的格式來計算，可以儲存至少100張CD的歌曲。透過USB與PC串接
後，使用PhatMan MP3管理程式，即可將下載MP3檔案至 PhatBOX，並
進行播放管理等動作。

而PhatBOX的核心，則採用了Cirrus Logic 公司推出的Maverick處理
器，並支援嵌入式Linux上網裝置。Maverick處理器整合了ARM晶片和
Crystal公司的音頻DAC﹙數位至類比轉換器﹚，同時這顆型號EP7212
system-on-chip﹙內建 Linux作業系統﹚處理器還搭配了8051微控制
器，不但可以連結操控一般車用音響，更可以提供良好的噪聲整形功
能，過濾雜音以提高音質。

PhatNoise公司的PhatBox汽車音響系統曾榮獲第三屆MP3 高峰會頒發
的最佳產品獎。據PhatNoise表示，目前PhatBOX支援格式除了 MP3與
Wave檔﹙WAV﹚，還有可將檔案壓縮至MP3 一半大小的Windows Media
Audio﹙WMA﹚。而 PhatNoise將持續針對其它格式的多媒體檔案格式
進行支援開發。

■ Philips發表兩款Linux平台資訊家電模型

沈美君/編譯
2001/01/08

飛利浦﹙Philips﹚在消費者電子展發表兩款資訊家電模型，擁有LCD
觸控螢幕，並相容於WindowsCE 3.0、Mobile Linux、Linux等系統平
台，協助並非在電腦運算產業的製造商，也能加入網路家電研發的行
列。

------

飛利浦﹙Philips﹚公司在日前美國消費者電子展﹙Consumer Elec-t
ronics Show﹚發表兩款即將完成、且適合網路家電使用的參考平台
﹙reference platform﹚，可協助製造商更容易進入網路家電市場。
飛利浦預計這兩款家電模型將在今年下半年度開始銷售。

Philips展覽的這款具有80％完成度的產品，擁有LCD觸控螢幕，相容
於WindowsCE 3.0、Mobile Linux、同時支援Linux等不同系統平台。
產品的模型之一使用Transmeta的Crusoe 處理器，另一款則採用國家
半導體發展的Geode晶片組。使用Geode SC3200 處理器的產品雛形，
適合於使用電池為主的小型家電，而另一款採用Transmeta的 Crusoe
TM3400處理器的模型，則適用於常駐網路線上以及視訊方面的家電。

Philips 研發此款產品的目的，是希望協助不在電腦運算產業的製造
商，也能加入網路家電研發的行列；同時，也希望轉變公司的角色，
將製造和研發的工作交給後端的製造商，而持有品牌的公司則負責行
銷和通路方面的事宜。

此舉也暗示硬體製造領域的某些轉變。過去零件製造商﹙components
makers﹚提供產品原件給給製造商，然後製造商完成產品後，再販賣
給銷售者大眾。漸漸地，設計和製造的工作往後交付給零件製造商以
及合約製造商，而持有品牌知名度的製造商，慢慢在組織中只擔任行
銷和通路的角色。

■ Linux 2.4.0釋出  Linux之父心情寫真

謝良奇/編譯
2001/01/08

經過長達一年的引頸期盼，Linux 2.4.0 新版核心終於新世紀正式釋
出。Linux 之父Trovalds在核心釋出後的一場訪談中，說明新版核心
的改進與外界對他交棒問題的關切。

------

延誤預定進度許久，Linux 2.4.0 核心終於在1月4日釋出，除了滿足
歷經長久等待的眾多Linux使用者之餘，Linux之父Trovalds亦得以藉
此放鬆緊繃心情。在一場訪談中，Trovalds除了說明新版核心的改善
外，更對Linux領導交棒一事表達看法。

Trovalds一開始即表示，Linux 2.4 釋出後，像是讓用戶針對自身需
要自訂系統、不再被單一廠商綁住、優良的系統安全等，都是不論版
本如何改變，Linux所代表的真正意義，這也是Linux相對其它作業系
統的獨特優點。

至於提到有關托瓦茲在核心釋出佈告中的自嘲口吻，外界認為托瓦茲
心情狀況抑鬱。然而托瓦茲說那只是他的幽默表現，實際上他相當樂
於享受參與Linux的發展。就Linux發展交棒一事，托瓦茲認為長久以
來自己只處理專案中感興趣的部份，其它如穩定版本維護、開發核心
維護等等則是由其他人負責，當專案逐漸成長，許多人也對專案有重
大影響，這即是外界為何談論交棒的原因，事實上卻早已發生。

托瓦茲說，比起被套上 「Linux先生」的標籤，他更希望能被看作傳
統的技術領導人及資深設計者，而他確信這將會實現。至於下一版核
心的計劃，托瓦茲表示此時尚未考慮，因為他或許要先來一場徹底的
狂歡。

本次新版核心演進可看出明顯的技術升級，2.4.x 核心的特點主要在
改善系統基礎，如記憶體管理的許多改善。另外，隨插即用 ﹙PnP﹚
、PCi基礎建設、hotplug﹙如USB、hotplug PCI等裝置﹚等等，新版
核心皆有改進，Trovalds指出，這些改善主要是為使Linux 成為堅實
的基礎系統。

■ 美國NSA釋出加強安全版Linux套件

謝良奇/編譯
2001/01/09

一向以高度秘密著稱的美國偵察機構─國家安全局，日前釋出一份高
安全性的Linux作業系統原型，在產業界的協同開發下，加強Linux系
統原有的應用軟體安全機制，作為該領域各別研究專案的整合。

------

作為其資訊保全任務的一部份，國家安全局長久以來連同電腦安全研
究社群投入在眾多電腦安全論題研究，認識到作業系統安全在高層級
安全性中伴演極關鍵的角色。此次釋出安全加強的 Linux，可作為該
領域各別研究專案的整合。

國家安全局認為，基於資訊機密性及完整性要求，終端系統必須實施
資訊分離以提供系統安全。不幸的是現存作業系統大多缺少實施資訊
分離的關鍵功能：命令存取控制，使得應用軟體安全機制易受攻擊，
讓惡意程式或瑕疵軟體輕易造成系統安全問題。

至於為何選擇Linux作為開發平台，國家安全局表示乃由於Linux不斷
成長的普及率以及其開放環境，提供展示安全功能實現於主流作業系
統的機會。此外，國家安全局釋出此套Linux，並非希望作為Linux的
完整安全解決方案，相反的，該系統只是作為範例，展示命令存取控
制如何加入Linux系統。國家安全局表示期盼透過Linux社群持續發展
該系統。

在 Network Associates、Secure Computing、Mitre等公司協助下，
在安全性有所加強的 Linux，以提供強大、彈性的命令存取控制架構
，解決應用軟體安全機制不佳問題。技術上採用Secure Computing的
型態強制框架﹙Type Enforcement Framework﹚，以分段方式保護應
用軟體及網路服務，連同一套以角色為基礎的存取控制元件，定義安
全策略設定檔案。

安全策略設定檔案可決定系統作業的動作是否允許執行，換句話說，
命令存取控制能限制包括超級使用者程序在內的任何程序動作。此種
安全機制具有相當彈性，可藉由系統設定，因應多種安全需求。釋出
的原型系統中已內含一般用途的安全策略設定範例。

這一套Linux原型系統採用2.2.12版核心開發，目前只支援x86架構，
並僅於紅帽 6.1作過測試。國家安全局表示因資源限制，尚未將其安
全機制最佳化。此外，系統授權方式採行GPL授權。

■ 可直接經Windows安裝的「Phat Linux v3.3」

沈美君/編譯
2001/01/09

Phat Linu可直接經由Windows，將Linux安裝於DOS的分割磁區上，在
1月8日推出的新版「Phat Linux v3.3 」，包含更多新功能，只需使
用滑鼠點選，就能順利安裝完Linux。

------

Phat Linux 於1月8日宣佈推出新版本「Phat Linux v3.3」，除了可
經由Windows來安裝Linux之外，更增加了許多新的應用軟體，和更新
程式。例如 KDE 2.0，內含Koffice辦公室套件；以及XFree86 3.3.6
、Gnome、Enlightenment、Apache、 Dos模擬器、音效程式、影像編
輯器、FTP伺服器等。

Linux 初學者或許最害怕的是安裝前的磁區分割，因為一不小心分割
錯誤，可能導致其他磁區的資料遺失。雖然現在已經有不少套件﹙例
如Mandrake﹚，在安裝步驟中，已經有圖形介面的分割、格式化工具
，但這對只想試試看Linux 上某些應用程式來說的使用者，似乎還是
不夠方便。

Phat Linux是相當特別的Linux套件，可經由Windows將 Linux安裝於
DOS分割區﹙DOS partition﹚，方便 Linux初學者不用經歷磁區分割
之苦，就能一窺Linux的全貌。

Phat Linux 3.3 版的windows安裝程式，包含更多新功能，只需用滑
鼠點選，就能順利安裝完 Linux。安裝完畢之後，將自動偵測硬體組
態，初學者在不需要手動調整的狀態下，就能開始使用X window了。

伴隨新版Phat Linux而來的，是更多應用軟體套件，如 KDE 2.0，內
含Koffice辦公室套件；以及XFree86 3.3.6、Gnome、Enlightenment
、the Apache網頁伺服器、Dos模擬器、音效程式、影像編輯器、FTP
 伺服器等。

Phat Linux v3.2 到v3.3的重大改變，是使用者可以自行選擇核心﹙
kernel﹚版本，Phat Linux 3.3所用的核心為 2.2.15版。

Agenda公司的總經理Brad LaRonde表示，Agenda已認知到數位及無線
技術將會轉變消費者溝通及組織其工作及遊戲的方式。而VR3 就是基
於滿足消費者這項需求而生的。

Agenda的VR3具備大型2.25" X 3.25"吋的LCD顯示幕，解析度160X240
，16灰階，採用66 MHz 32-bit MIPS處理器﹙大約是同價格等級設備
的4倍處理能力﹚，並且以Linux為作業系統。使用者將可以執行多工
作業，亦即在收送電子郵件的同時，使用者還能使用其他應用軟體。

Agenda VR3並附有QuickSync軟體，使得VR3上的應用軟體將可以與PC
端的應用軟體藉由業界標準的紅外線通信協定相互交換資料。標準配
備8MB隨機存取記憶體以及16MB的快閃記憶體，以及其他輸出、輸入
功能，整台VR3的售價僅美金199元。

VR3之所以能夠將這麼多的功能擠進一個小盒子中，要歸功於NEC的VR
4181晶片系統，在VR3中扮演心臟的角色。VR4181原本是NEC專門為微
軟WindowsCE開發的PDA晶片，Agenda香港母公司在經過一番評估之後
，決定朝向開放並易於開發的應用軟體平台發展，最後他們決定採用
Linux，並將VR3設計成開放式架構。

Agenda決定不採用專屬系統的技術，也希望能吸引開放系統社群的人
加入在VR3上的軟體開發行列。Agenda 並已經能夠利用既有的開放原
始碼計劃，豐富在VR3上的資源。

■ 開放原始碼的文書軟體--AbiWord新版釋出

馮文龍/編譯
2001/01/08

桌上型電腦系統之所以普及，在於有很好的工具軟體支援，但目前最
普及的辦公室工具軟體幾乎都在微軟的視窗作業系統之上。因此開放
原始碼計劃裡的Abiword 專案或許是個不錯的另類選擇。

------

AbiWord是由AbiSource公司所主持的開放原始碼專案計劃。這個計劃
的目標聽起來頗讓人覺得望之卻步：AbiWord打算建立一套跨平台、
的文書處理系統。

但如果只是這樣還不算是什麼雄心壯志，AbiWord將有支援Linux的版
本在Intel、Alpha、PPC，以及BeOS版本在Intel及PPC 上，另外還有
不同的版本支援Unix、QNX、以及Windows95/98/NT、MacOS等。

最近發表的AbiWord是版本0.7.12，並且獲選為GNOME的辦公室工具套
件，透過GNOME的龐大技術支援體系，AbiWord將可快速地得到支援。
AbiWord的出現，正意味著使用者將可僅靠一招半式就可闖蕩江湖。

功能上，AbiWord包括了基本的字型格式、段落格式、拼字檢查、RTF
檔案格式以及Word97 的匯入功能，並能匯出成RTF、文字檔、HTML以
及LaTEX 檔案格式，另外交談式的尺規及定格設定、無限制的動作復
原／重作、風格設定、尋找／取代、影像格式等特色。

整體來說，AbiWord 將非常適用於每一到兩週需迸出一到兩篇信件的
使用者，至於需有大量生產力的需求，例如寫書或高檔次的報告，則
可能還有改進的空間。

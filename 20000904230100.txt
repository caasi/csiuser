■IBM在亞洲投入2億美元發展Linux

蕭喜那/編譯                                                                              
Sep 01 2000

IBM 宣布未來的四年之內，將投入二億美元以上的資金，在亞太地區
實施Linux 倡導方案，以使該公司的亞太顧客與軟體販賣商可以更快
適應Linux作業環境。方案的主要內容，包括：成立Linux開發中心、
Linux稱職中心﹙Compentency  Centers﹚、與Linux企業夥伴的結盟
等。

------ 

IBM宣布將於亞太地區實施Linux倡導方案，主要目標在於快速提升亞
太顧客與重要的軟體販賣商的能力，以增加Linux 作業環境下的生產
力。

未來的四年之內，IBM 在亞太地區將投入二億美金以上的資金，主要
的項目包括：七家 Linux 開發中心、Linux 稱職中心﹙Compentency
Centers﹚、與Linux企業夥伴的結盟、Linux 研究與開發及其他程式
、布署三百位以上專業的Linux 顧客、硬體與軟體的專家、研究員、
產品開發人員以及專業的服務人員。

IBM在亞太地區即將成立的Linux 開發中心，可說是此次Linux倡導方
案中最重要的一部分，目標在輔助軟體販賣商將他們的應用軟體接到
Linux 的環境中。這七家中心將位於東京、上海、北京、台北、漢城
、Bangalore以及雪黎。Linux的開發人員可以與這些中心聯繫，並且
可以透過IBM為亞太新成立的網站﹙www.ibm.com/linux/ap ﹚，存取
其他的Linux資源。

IBM的Linux倡導方案中，很重要的部分是：該公司的developerWorks
﹙www.ibm.com/developerWorks﹚網站將出現日文與中文的版本，這
是IBM為開發社群所設立，獲獎的、完整的線上資源。

為了加強對Linux開發人員的支援，IBM也將在日本大和、北京、漢城
、Ban-galore設立Linux稱職中心。這些中心將於Linux社群合作共同
開發標準，例如Li18nux、Motif Globalization、GNU  Text  Tools
Unicode supoport等。

IBM在亞太市場的合作夥伴包括Intel、TurboLinux等。TurboLinux是
世界上最大的Linux作業系統的開發與經銷商之一。

■昇陽會不會支持「OSDL」？

黃翔祺/編譯                                                                     
Sep 01 2000

英特爾、IBM、NEC等電腦大廠上週三宣布成立開放原始碼發展實驗室
﹙OSDL﹚，而一向與開放原始碼社群關係良好的昇陽公司，這次沒有
加入實驗室的開發，卻另外成立一個組織─  Gnome 基金會﹙ Gnome 
Foundation﹚，於是有人開始懷疑，OSDL計劃的頭號對手會不會就是
昇陽？

------

經過一個暑假的籌備，8月30日數家硬體大廠，包括英特爾、IBM、
HP及美國NEC，宣布將成立一個開放原始碼發展實驗室﹙Open Source
Development Lab，OSDL﹚。

除此之外，該實驗室也與主要的Linux 廠商發展策略聯盟的關係，這
些廠商包括： SuSE 、 紅帽 、Caldera 、 TurboLinux、LinuxCare
等Linux軟體暨系統廠商，及Linux嵌入式系統廠商LynuxWorks。

目前，該實驗室的運作方式是，每家大硬體廠商都有自己的實驗室，
及執行長，但是，這些實驗室利用網際網路與 OSDL 其他實驗室成員
進行互動，並將成果發表至網際網路上。

值得注意的是，過去與開放社群關係一向友好的昇陽公司並未加入該
組織。但該組織已經公開澄清，表示歡迎其他廠商的加入合作，昇陽
公司總裁當時表示：「由於該實驗室的條件是開放原始碼，而昇陽公
司的主力產品為Solaris ，是不開放原始碼的。」

但是後來事情有了戲劇性的變化，昇陽集合上下游廠商與 Linuxcare 
推出另一開放原始碼組織─Gnome 基金會，讓事情的發展形成一個有
趣的弔詭。看來昇陽有意自立門戶，與OSDL一別苗頭的味道。

■紅帽Linux成長率居冠 稱霸全球市場

張書銘/編譯                                      
Sep 01 2000

隨著電腦的普及化以及技術變得容易取得，想要像過去一樣在軟體市
場上獨霸一方，已經愈來愈難，在作業系統上尤其如此。過去被認為
是少數技術人員才會使用的Linux 及其延伸出來的系統，漸漸在市場
上揚眉吐氣。

------

今年紅帽Linux的市場成長率，以69%拔得頭籌。而紅帽能有如此的好
成績，和它易學易用不無關係，在目前的Linux 套件當中，紅帽提供
良好的商業支援服務，而且軟體安裝簡單，是它能夠成就高成長率、
被認為是Linux 商業界龍頭的主因。

次於紅帽的，是軟體國際化的先驅─德國Linux供應商 SuSE，在歐洲
方面獨霸一方是它能夠緊追紅帽其後的主因。而其他諸如   Caldera 
System和TurboLinux等等也都有很好的表現。

不過，雖然目前各家Linux 的市場競爭激烈，大致上仍舊保持井水不
犯河水。舉例來說，紅帽不會刻意去吞下 OpenLinux專擅的網路伺服
器市場，而OpenLinux也得以在這領域裡保持一定生存空間。

根據IDC﹙International Data Corp.﹚去年度的統計， 1998到1999
年之間，整體Linux的市場成長了89%。在這樣的環境之下，大家都不
得不承認Linux確實成為了微軟和其他Unix─例如昇陽的Solaris之外
的一個更好的選擇。也因此，在Linux 界當中，各家廠商的競爭也相
對的激烈起來。

儘管在電腦業界Linux已經是老面孔了，但在商場上，Linux仍有著極
大的開發空間。在伺服器市場上，Linux佔有率已達到25%，僅次於微
軟Windows的38%。雖然多數都還只限於低階的伺服器，但仍然是不可
小看的，因為這並不表示Linux 在高檔應用上沒有商機。相反的，它
不但可以做各類單一功能的低階伺服器，更能透過結合其他商業界需
求的軟體解決方案。因此，Linux必定還有更寬廣的路可走。

■Java-Linux平台的嵌入式設備  可望撼動Wintel霸權

謝良奇/編譯                                 
Sep 01 2000

面對漸趨複雜的新一代嵌入式設備， Linux及Java兩者的特殊優勢，
除了將為該類型設備提供極為適合的作業平台，利用Java，嵌入式設
備能與後端伺服元件互動，執行各項功能，將撼動現今Wintel霸權。

------

Java平台及 Linux作業系統由於其獨特的優點，未來出現在嵌入式應
用環境中的機會將更為頻繁，Java-Linux的組合式平台，更得以提供
開發人員，加速設計流程、降低成本等優勢。而 Linux、Java、嵌入
式系統三者相互結合的背後，更隱含象徵Wintel霸權時代的終結。

Linux作業系統，近來在嵌入式設備應用上受到矚目，像是 Lineo 、
LynuxWorks及MontaVista等公司，皆相繼發表各自的嵌入式Linux 系
統。對嵌入式系統來說，Linux 具有數項關鍵性優勢：無執行權利金
、開放原始碼、高移植性、可配置的核心等等。

此外，僅管Java一度因技術定位及執行速度等等問題而遭受挫敗，但
其物件導向、高移植性、良好的記憶體管理及網路支援等特點，十分
符合嵌入式系統的需求。Java技術的最大優點在於，伺服器物件的可
重用性，其 Swing GUI元件豐富先進的界面，則相當適用於各式嵌入
式設備，使得Java在嵌入式應用上前景看好。新的J2EE﹙The Java 2
Platform, Enterprise  Edition﹚即在於為如Web伺服器、郵件伺服
器、目錄伺服器、資料庫及即時傳訊系統等各式元件，提供標準化的
Java語言界面，適合作為嵌入式設備的應用軟體平台。

與其它作業系統相較，採用 Linux能為使用者節省大筆費用。在持續
不斷的改進之下， Linux預料將大幅擴張嵌入式設備領域的現有版圖
。值得注意的是，由於 Linux系統的普及度與其原始碼的快速散布，
Linux 預料將移植到各式微處理器，如此一來英特爾獨佔的處理器霸
權勢必遭受挑戰。

其實嵌入式設備、 Linux及Java三者，就像是即將合而為一的開放自
由勢力，藉由Linux 系統的普遍移植，各式設備將不再受限於英特爾
的微處理器，利用Java，嵌入式設備能與後端伺服元件互動，執行各
項功能，將直接搖撼現今以微軟為主，桌面應用環境的軟體霸權。

■波音公司推出Linux系統的空中網路服務

張書銘/編譯                                      
Aug 30 2000

隨著網路化的力量打入社會每個角落的風潮，附加網路服務成為一種
吸引客戶上門的手段。經過今年四月間的內部總體檢之後，航空界的
巨人波音公司終於決定推動航班上的網路服務來爭取在航空界不敗的
地位，並決定採用Open Source的Linux系統。

------

今年四月，波音公司為了檢視公司的競爭力，為自己做一次體檢。在
過程中，波音公司對自己的電腦體系也做了評估。在參照了知名的五
百家企業經驗後，波音公司對於Open Source的Linux系統表示了相當
的肯定。而在提升網路上的能力及服務，則成為波音公司努力的方向。

基於商務旅行的客戶愈來愈多的考量，波音公司必須提供適當的通訊
服務才能留住客戶。根據波音自行評估的數據顯示，未來的二十年間
，整個航空市場會比目前成長兩倍。而這許多的旅客勢必會有收發電
子郵件、瀏覽網頁，甚至是開視訊會議的需要。因此波音公司決定替
飛機做新的配備，配合衛星系統提供網路服務。

舊有的航空網路服務常因為品質不佳而被認定麻煩、不切實際，而波
音公司採用新的天線以及系統，大幅提升了無線網路的效能。下載速
度可達到5Mbps，而上傳速度也有1.5Mbps，比起一般的T1專線來可說
是毫不遜色。而在軟體的配合上，除了配合一般使用者的習慣而使用
了Windows 2000外，也提供了Linux讓使用者有選擇的空間。

目前這項新的無線網路服務還只限於波音公司在美國本土內的二十條
國內航班。而相較於一般航空電話平均每通13美元的要價，平均寄送
一封電子郵件只花4.95美元的低價已經引起許多顧客的使用興趣。因
此波音公司也已經開始與日本及義大利方面連繫，希望在這兩地設立
維護及管理的系統以便將無線網路的服務推展到國外航線。

■以ASP模型來發展的辦公室套件，是好主意嗎？

沈美君/編譯
Setp 01 2000

ThinkFree公司所發展的「ThinkFree Office 」辦公室套件，以 ASP
模式進行，企業可免去購買和維護應用軟體的成本，使用者只要透過
可連接網路的設備，就可以使用辦公室套件。目前正在開發中的版本
有Windows、Linux、及 Mac，免費無功能限制地開放使用，而額外付
費則可獲得較大網路硬碟空間，以及其他備份服務。

------

八月下旬舉行的Linux World 盛會，大部分的焦點都聚集在「 Linux 
的桌面環境發展」，尤以 Gnome 和 StarOffice 的合作，被譽為 MS
Office系列的首要潛在威脅者。不過，ThinkFree公司以ASP模式所發
展的ThinkFree Office，則充分利用網路無所不在的特性，提供企業
多元的辦公室套件。

ASP﹙Application Service Provide﹚是一般泛稱的「應用程式服務
提供者模式」，也就是使用者必須透過網路，由伺服器來取得／執行
應用程式，不必限定於單機操作，檔案也可儲存於廠商提供的網路硬
碟，大大提升了工作地點的彈性，企業亦能免去購買和維護應用軟硬
體的成本。

ThinkFree公司以ASP架構，發展「ThinkFree  Office 」辦公室套件
，目前包括文書處理、試算表、簡報等功能。ThinkFree 的總裁 Ken
Rhic表示，ThinkFree Office以微軟Office系列的全功能為目標，以
Java Applets來發展ASP辦公室軟體，但不是以打倒 MS Office 為主
要策略，而是讓使用者能夠透過 ASP的特性，隨時隨地都能上網使用
ThinkFree Office來編輯與MS Office相容的文件。

ThinkFree Office  主要透過 Java Applets 運作，使用者第一次執
行時，只要到 ThinkFree  站註冊並下載 Applets，每個模組約只有 
1.5MB 大小，若偵測到使用者的 Java 版本並不適當，將會自動更新
。ThinkFree Office  目前比較大的問題，不是各種平台對 Java 支
援的差異，而是同一種平台上，有不同版本 Java 之間的差異。

在檔案的同步化處理方面，ThinkFree Office  在目前是使用類似檔
案管理員的手動處理方式，讓使用者在網路硬碟和本地端電腦硬碟間
，作檔案的複製、搬移等處理，未來將新增類似 Palm Pilot 的自動
同步化功能，確保使用者在網路硬碟及本機電腦檔案的同步性。在安
全性問題方面，ThinkFree Office  預設的安全機制只有登入帳號／
密碼，但允許使用者加入其他的安全機制，例如將檔案加密。

ThinkFree 計畫在十一月發表可在傳送過程加解密檔案的功能，未來
將繼續發展直接在ThinkFree Office中執行加解密，並能儲存網路硬
碟裡。目前，ThinkFree Office為免費全功能地開放下載使用，附10
MB的免費網路硬碟空間，但是必須接受廣告。若一年付費25或99美金
，將可加大網路硬碟空間，以及更久的備份復原處理。

■Linux 3D支援漸增 遊戲軟體不再缺席

藍雲紋/編譯                                
Sep 01 2000

來自各廠牌3D卡的支援大幅提昇。遊戲軟體從PC版，擴展到Linux 版
。讓Linux在遊戲軟體界，不再是缺席者。

------

由於來自硬體的支援持續增加，兩年前只有一種卡有支援 Linux，現
在則幾乎對所有3D卡都支援了。有些公司甚至製作他們自己的 Linux
埠，讓玩家可以玩更多的遊戲。

LokiGames 公司將許多個人電腦上的遊戲，轉換到Linux 系統。他們
確定，使用它們的LGPLed安裝工具，就可以將遊戲輕易地安裝。開放
原始碼社群也生產了一些有價值的遊戲，並且完全免費提供。Loki對
Linux的貢獻還不僅於此，它還幫助其他研發者轉移到Linux來。並且
自行發展出一套研發者工具。

除了Loki之外，Indrema也是參加此次Linux世界展中，研發遊戲軟體
的廠商。展出的硬體雖是借來的，終究能讓參觀者一窺遊戲世界的未
來，但可惜的是，沒有研發者能確定系統進一步的研發。Indrema 當
然了解開放原始碼的所帶來的威力能讓它們與微軟等大公司競爭。開
放原始碼，改革電視遊樂器工業，將遊戲平台的擁有權轉移到研發設
權，讓他們能促使平台更快更遠，讓遊戲軟體更多、更好。

不過，儘管有硬體與軟體的支援，Linux 還不是完全可行的遊戲平台
。Linux 的遊戲應用，必須依賴桌上型電腦的成長。硬體代理商的支
援不足，讓第三方程式設計師在製作驅動程式時，對硬體規格的知識
不足。3D 圖像的支援，也應統一化。

專家表示，明年可望有更多目前玩Windows遊戲的使用者，轉向Linux
，讓 Linux遊戲在桌上型電腦的市場佔有率能明顯提昇，甚至超越麥
金塔，成為桌上型第二名的作業系統。

■Linux－PDA的市場新貴

馮文龍/編譯                     
Sep 01 2000

一個稱為Linux VR的專案正在計劃將Linux帶入PDA市場，該專案開宗
明義標示著：我們的目標是將Linux 作業系統引進NEC VR系列處理器
的設備上。

------

大多數的VR系列的裝置原本都是被設計來執行Windows CE的，而現在
Linux VR專案已使得Linux得以在Vadem Clio，Casio E-105，Everex 
Freestyle等設備上執行。

除了VR系列的設備外，開放視界電子報不久前也曾介紹過韓國三星電
子推出的Yopy，這款也是以Linux為平台，並採用ARM 206MHz RISC處
理器的PDA，整合了許多的多媒體應用，如MP3、數位攝影機等。另外
Agenda電腦公司也在LinuxWorld公告上市一款以VR系列處理器為主的
PDA─VR3。

Agenda VR3採用 NEC 66MHz的MIPS VR處理器，在功能上強調Linux作
業系統本身的多功能力，以及網際網路的連線能力。在隨機存取記憶
體的搭配上，Yopy搭配了32MB的大容量記憶體，VR3則只搭配8MB的記
憶體，在LCD顯示幕上，Yopy搭配240x320高解析度TFT顯示幕，VR3則
是160x240單色LCD顯示幕。我們從Yopy的零售價大約在400至600美元
，而VR3則鎖定在149到299美元之間大約可以了解到兩款PDA在配備及
功能上是有些差異的。

韓國三星的Yopy是由GMate設計，VR3則是完全百分之百相容於VR專案
。三星電子相信大家都不陌生，而Agenda電腦公司的重心集中在 VR3
、VR5 及未來可攜式電腦設備的行銷，以及客戶服務上，是由以香港
為基地的電子製造業巨人Kessel Internation Holding所投資。

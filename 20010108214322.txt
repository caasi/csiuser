■ Linux kernel 2.4正式上菜！

沈美君/編譯
2001/01/05

讓產業界及Linux愛好者等了一年多的 Linux 2.4 心正式版，終於釋
出了。Linux 2.4 kernel新增對稱多工處理的支援，適合應用於高階
企業，並支援 USB，將更適合家用網路設備及個人桌上平台電腦，可
經由FTP下載新版核心。

------

Linux kernel 2.4 核心測試版才問世，Linux之父托瓦茲於1月4日時
，由 Kernel Mailing List﹙linux-kernel@vger.kernel.org﹚發出
一封主題為「And oh, btw..」的信件，宣布 Linux kernel 2.4核心
正式版釋出的消息。

2.4新版核心一向被視為Linux運動的重要里程碑，Linux 主要領導廠
商，例如紅帽、Caldera、德國SuSE公司等，都相當期待2.4版核心的
新特色與能力。久候一年，原本應該在去年秋冬季就釋出的核心，終
於正式推出。

「夠了就是夠了，經過同樣一批人測試再測試，應該也不會有更好的
改進，」托瓦茲發給mailing list的信件裡以風趣的口吻表示，因為
發展者們過度疲累，只好簡單用「應該比前一個版本核心好」來形容
2.4版核心。

Linux 2.4 kernel新增對稱多工處理的支援，適合應用於高階企業的
Linux伺服器及應用軟體，以往的Linux核心最多只能支援四個處理器
，新版核心最多可支援十六個處理器來做多工運算。此外因為增加對
USB的支援，2.4版核心將更適合家用網路設備及個人桌上平台電腦。

■ SuSE進軍掌上型電腦市場

馮文龍/編譯
2001/01/05

SuSE近期將積極進軍掌上型電腦的嵌入式作業系統市場。 而Linux則
似乎已成了下一代掌上型電腦的新寵，類似像康柏電腦的iPaq等都在
打Linux的主意。

------

SuSE打算利用這一波潮流切入嵌入式作業系統市場，並以掌上型電腦
作為跳板。但想要在這個市場成功，除了一個可行的營運模式外，還
必須尋求可靠的夥伴關係，而SuSE過去則並未投入太多心力在這一個
市場。

現有的個人數位助理市場以Palm OS及微軟的Packet PC為主流。而著
名的市場研究機構IDC系統軟體研究副總Kuznetzky表示，由於 Linux
的開放原始碼特性，及易於開發應用系統及執行伺服器軟體使 Linux
有如黑洞般吸引嵌入式系統廠商加入Linux的行列。

目前僅有少數的製造商生產以Linux 為主的掌上型電腦。韓國的三星
電子在去年的五月發表了Yopy個人數位助理，而設置了生產線的廠商
則包括了惠普科技、康柏電腦、IBM等。

對於SuSE決定加入這個戰場， Kuznetzky預期：SuSE將有一段艱苦的
路要走，但Kuznetzky卻也認為， SuSE將不能再放過進入這個行動式
計算機市場的機會。

Caldera的一個子公司Lineo已在這個市場上奠定了穩定的基礎，而紅
帽也有不錯的成績，而SuSE則僅佔這個市場的9.5％。 但儘管如此，
其他的作業系統則最好注意，因為Linux 已經加入了這一個競技場，
預估將會有大量的廠商投入屬於開放原始碼的Linux的懷抱，Linux似
乎已成了下一代掌上型電腦的新寵。

■ 日立推出以全美達Crusoe為基礎的網路應用裝置

謝良奇/編譯
2001/01/05

日立公司日前宣佈，將在日本銷售以Crusoe為基礎的新款網路應用裝
置Flora-ie 55mi。該款裝置採用Linux系統，一舉結合筆記型電腦、
掌上型裝置、行動電話功能，但因採用全美達的新處理器Crusoe，該
處理器耗電量低的特點，Flora-ie55mi可兼顧強大功能及省電優點。

------

日立公司﹙Hitachi﹚於去年底對外宣佈，將在日本銷售以 Crusoe處
理器為基礎的網際網路應用裝置，名為Flora-ie55mi。此新款網路應
用裝置，主要採用全美達﹙Transmeta﹚以低耗電量著稱的 Crusoe處
理器，除直接於日本銷售外，日立亦計劃在美國推出此款新型裝置。

Flora-ie55mi是結合筆記型電腦、掌上裝置、行動電話的多功能網路
應用裝置。該裝置外觀上有一個如筆記電腦的螢幕，可直接連接網路
並搜尋網頁或執行應用軟體，透過如Palm一般的筆式界面，可輸入資
料。此外，Flora-ie55mi更內含行動電話界面，可供使用。因為採用
低耗電的Crusoe晶片，電池使用時限長達七小時之久。

特別的是，Flora-ie 55mi和去年11月由美國線上、蓋特威﹙Gateway
﹚推出以 Crusoe為基礎的網路終端設備一樣，皆採用Linux系統，但
日立公司與全美達兩家公司本身的筆記型電腦卻選擇微軟的視窗系統
。

全美達執行長Dave Ditzel表示，該公司旗下的 Crusoe處理器，讓日
立得以在掌上裝置內加入個人電腦的所有特性，卻仍維持長達七小時
的電池時限。

此外值得一提的是，以行動網際網路計算市場，開發平台解決方案為
主的全美達，在推出宣稱耗電量低於Advanced Micro Devices、英特
爾等競爭對手產品的新處理器─Crusoe後，近來在市場上紛擾不斷。

首先在公開上市前，IBM 宣佈中止開發內含Crusoe的ThinkPad，隨後
表示將改採英特爾的低耗電Pentium III。其次是NEC日前因產品內含
晶片有部份瑕疵，回收以全美達晶片為基礎的筆記型電腦，當時全美
達股價曾一度下挫。

■ LinuxCare發起Linux Plug Fest計劃

謝良奇/編譯
2001/01/05

為了整合未來將出現的硬體架構技術與新的Linux核心，LinuxCare公
司發起一項 Linux Plug Fest活動，號召硬體廠商與軟體開發者一同
參與，加強未來 Linux核心與其他軟硬體技術的結合，同時也讓技術
尚未成熟的OEM廠商能夠獲得充分的支援，提昇彼此在Linux方面的能
力。

------

成立於1998年的 LinuxCare是一家提供企業延伸科技服務的公司，同
時也進行招集開放原始碼開發社群的工作。為了能讓未來 Linux核心
對於不斷推陳出新的硬體架構與軟體元件有更高相容性， LinuxCare
決定推動一項稱為Linux Plug Fest的活動，提昇各類Linux產品的品
質。

這項活動主要邀請OEM廠商以及原件開發社群的人士，於1月21到23日
舉行。在首次的 Linux Plug Fest活動中，將以主機板及BIOS的供應
商為主軸。整個活動將由軟硬體的廠商與社群提供它們的新產品，並
由LinuxCare來提供Linux系統進行測試，並安排科技顧問在現場提供
協助與分析，並製作結果報告，讓廠商做為改良的參考，同時也有專
題研討，主題包括OHCP﹙Open Hardware CertificationProgram﹚與
Linux及開放原始碼的開發特徵。

由於這類型的活動是第一次舉辦，因此著重於提供代工廠商技術支援
的層面，由LinuxCare釋出的活動資料顯示，這項Linux Plug Fest主
要將訴求過去沒有 Linux開發相關經驗，或軟體技術不足的代工廠商
將想要用於Linux的產品提供出來，由LinuxCare來提供技術支援。

LinuxCare 公司過去曾為各大廠商進行類似的測試工作，例如惠普公
司的NetServer系統針對包括Caldera、SuSE、TurboLinux和Debian等
Linux產品的測試和驗證工作就是由Linuxcare負責進行。因此對於代
工性的廠商而言，這次的活動將可自 LinuxCare方面吸收到不少大廠
的經驗。

■ 飛利浦USB控制介面獲Linux支援

馮文龍/編譯
2001/01/04

SoftConnex科技公司與 ScanLogic以及飛利浦半導體分別針對通用序
列埠﹙USB﹚的解決方案簽訂合作協議。SoftConnex的USBLink軟體將
與ScanLogic的主從架構控制器SL11H/SL811HS整合，而飛利浦半導體
的ISP1161/2單晶片USB主機以及控制介面，與 SoftConnex的USBLInk
軟體將整合成為一個USB主機的解決方案。

------

SoftConnex與ScanLogic的合作，在於提供OEM廠商一個完整的嵌入式
系統的USB解決方案。SoftConnex的USBLink軟體將與 ScanLogic的主
從架構控制器SL11H/SL811HS整合，使得像是PDA、網路設備、數位攝
影機以及資訊家電等設備能夠與USB的週邊設備通訊。

飛利浦半導體與SoftConnex的合作則是將飛利浦半導體的 ISP1161/2
單晶片USB主機以及控制介面與SoftConnex的USBLInk軟體整合成為一
個USB主機的解決方案。ISP1161/2可同時扮演控制介面與主機的功能
而USBLink則支援多種嵌入式作業系統，包括Linux。

USBLink 可提供OHCI以及／或是UHCI嵌入式軟體驅動程式，同時支援
多種作業系統，包括Linux、PowerTV、VxWorks、Nucleus、 ThreadX
、QNX以及其他等。

USBLink創造了一個開發類別驅動程式的架構以支援USB設備。而這個
架構能夠允許以類別驅動程式管理 USB設備的組態設定或以 USB的堆
疊﹙stack﹚來執行必要的組態設定、介面初始化以及USB資料管道或
端點的管理。

USBLink可支援單一USB設備連接主要集線器的簡單拓樸，也支援類似
多集線器對多USB設備的複雜拓樸。USBLink在主機的RTOS中模擬一個
隨插即用的環境，使得使用上更為便利。

在微處理器上，USBLink支援像是Intel x86、ARM、StrongARM、MIPS
以及PowerPC等。同時提供了類別程式庫給各類的USB設備使用，像是
鍵盤、硬式磁碟機...等。

生。

應用軟體服務提供商﹙ASPs﹚與網路服務提供商﹙ISPs﹚廣泛地使用
專業伺服器，包括Web主機與資料庫主機裝置等。 EBIZ在這個領域的
競爭對手包括Cobalt、Network Engines、與Network Appliance等。

2001/01/04

TelcoSoft近日推出Extensys 1.0.2版，可在Linux x86、SCO Unixwa
re、與 AlphaTru64 UNIX等作業平台上運作，Extensys可供開發人員
開發與布署通訊／電信應用軟體。TelcoSoft並在網站上提供Extensy

------

TelcoSoft公司近日推出 Extensys 1.0.2版，它可以支援64位元架構

信網路﹙經由電話﹚或是服務提供者的網站，存取由這些應用軟體所
提供的服務。

為了鼓勵開發人員使用Extensys開發通訊／電信應用軟體，TelcoSof
t在該公司的網站上提供 Extensys的免費下載版本。但是，這個免費
版本只限於開發目的專用，商業使用目的則另有一個獨立的授權許可
。免費Extensys的下載具備完整的功能性，除了它限制語音埠的數目
，只能供四個應用軟體所用。免費版本的授權許可在 180天以內有效
，之後，可以到 TelcoSoft的網站上重覆更新。此項更新可以確保使
用者擁有最新版的Extensys。

TelcoSoft 也計畫提供數個熱門電信應用軟體的免費下載。這些應用
軟體採用開放原始碼模式，也就是任何人都可以免費使用這些軟體的
修正或原始的版本。目前，在TelcoSoft的FTP站上，有兩項免費的軟
體可供下載，即 E-Mail Reader，可以讓使用者在電話上聽取電子郵
件訊息；以及郵寄名單應用軟體。

■ 3D遊戲F.A.K.K.2將轉戰Linux平台

張書銘/編譯
2001/01/05

3D動作冒險遊戲 Heavy Metal F.A.K.K.2﹙中譯：重金屬女戰士﹚即
將搬上 Linux平台。這款開放原始碼的遊戲，憑著驚人的3D技術以及
狂野的重金屬音樂吸引了不少年輕玩家。而這次移植到 Linux平台上
的決定，也顯示Linux的相關繪圖技術，已受到開發者的肯定。

------

在去年美國E3展中讓許多玩家驚嘆不已的 PC GAME重金屬女戰士，即
將移植到 Linux平台，這款遊戲不但大量使用了重金屬音樂，同時也
擁有驚人的3D效能。重金屬女戰士是去年推出在 PC以及Dreamcast主
機上的遊戲，而且在推出後也公開它的原始碼及部分相關開發工具，
讓玩家能深入了解它的核心並創造遊戲關卡。

它採用了知名遊戲「雷神之鎚三」的遊戲引擎，以表現出精緻豐富的
3D效果，主角必須在豐富的即時運算3D場景中解迷並打敗敵人，才能
生存下去。而這款遊戲中最驚人的技術，就是在隨處可見的光影特效
。無論場景建築的特效光源；或是使用武器時的火光；甚至是敵我雙
方血肉橫飛的特效，全部都大量應用動態光源與即時運算的技術，讓
玩家能有身歷其境的感受。

同時在遊戲的功能選單中可以找到許多關於畫面控制的選項，包括像
頂點光源或線性光源這樣的差異都可以調整，充分顯現款遊戲在3D動
畫運算核心上所下的苦心。再加上這款遊戲走向重金屬的搖滾風格，
因此不只有一般的遊戲特效音，也有重金屬的背景樂，因此 Linux玩
家絕對可以充分感受到不遜於一般次世代遊戲主機的痛快感。

不過Loki公司也提出警告：由於這款遊戲改編自歐美的成人漫畫雜誌
，同時打鬥畫面也相當多，因此不適合較低年齡的玩家。

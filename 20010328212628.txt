■ SuSE大舉進攻企業伺服器領域

梁天佐/編譯
2001/03/27

SuSE Linux 7.1版日前大受好評，在 Linux市場佔有率領先紅帽之後
，SuSE乘勝追擊。SuSE為了在企業伺服器應用領域穩固發展，26日於
漢諾瓦CeBIT展中正式宣佈推出Linux企業級伺服器版本；SuSE Linux
7.1的PowerPC版本也將於4月初推出；稍早 SuSE在英國剛剛成立分公
司，主要也是鎖定企業市場。

------

SuSE公司26日在德國漢諾瓦 CeBIT展中宣佈，正式推出將穩定性與安
全性發揮到最大限度的SuSE Linux Enterprise Server，並透露支援
PowerPC的SuSE Linux 7.1將於4月初正式登場；SuSE稍早於英國成立
分公司，目標亦鎖定企業級市場，一連串積極的動作顯示企業伺服市
場的潛力與SuSE的決心。

SuSE Linux Enterprise Server 內含歐洲頂尖Linux專家為伺服器應
用開發的作業系統串流設計，作業系統內除了包含現有所有的伺服端
服務之外，也特別針對安全性與穩定性作最佳化的調整，不僅當作電
子郵件、網際網路伺服器的可靠性高，也可以在不同作業系統間處理
檔案與列印服務，由於防止當機的功能強大，更是要求嚴苛的應用程
式伺服器最佳的選擇。

另外，PowerPC版的SuSE Linux 7.1也將於4月初推出，最大的特色在
於支援內建 USB介面與PowerMac系列電腦的ALSA音源架構，SuSE將最
新版核心2.4.2針對PowerPC架構最佳化，同時調整超過一千款應用程
式以供Mac使用者應用，支援相當周到，可說是PowerPC中，最為完善
的Linux版本。

SuSE表示，SuSE Linux Enterprise Server 會在4月中正式出貨，完
整的內容包含三片CD-ROM與一本三百頁的手冊，PowerPC版Linux 7.1
則內含六片CD-ROM與一本五百頁的手冊，除了提供限期安裝支援外，
SuSE也對企業級使用者提供以個別電腦為單位，十分經濟的付費維護
服務。

在英國設立分公司也是針對企業級市場。英國在開放原始碼的企業級
應用上發展較為保守，SuSE英國分公司主要扮演諮詢的角色，不只支
援服務 SuSE自己發行的Linux版本，也支援服務有需要的企業所用其
他的 Linux版本，SuSE認為，引導企業應用開放原始碼軟體是一種人
人為我，我為人人的合作，一但企業開始使用 Linux相關的解決方案
，Linux就有機會被人更加了解，大家的機會都會變大。

SuSE英國分公司經理Michael Armstrong在接受Linux Today的訪談中
便指出：「在英國，我們傘狀組織的標題正是『專業的服務』，它包
含諮詢服務、技術服務與訓練，其中諮詢服務提供適當的企業級諮商
服務，加值的服務是非常重要的。」

■ IBM將以z/OS提振大型主機市場地位

謝良奇/編譯
2001/03/26

IBM 宣佈將推出大型主機作業系統z/OS，此系統將整合64位元模式等
多項功能，當昇陽等Unix伺服器廠商逐漸搶佔高階伺服市場時，z/OS
與Linux的支援將成為IBM重建大型主機地位的兩項主要策略。

------

IBM 宣佈將推出整合64位元模式特點的z/OS作業系統，這一套前身為
大型主機z900上的作業系統，將作為IBM 重振大型主機地位主力。 

z/OS前身是大型主機S/390的作業系統OS/390。z/OS 將提供多項功能
，其中之一是64位元定址的儲存空間能力，IBM 表示，往後記憶體系
統、資料庫交易的速度比起老式32位元定址將快上許多。除此之外，
z/OS更包含能迅速為作業轉移資源的智慧資源目錄及更完善的Java、
Linux支援。IBM承諾z/OS會讓大型主機更容易設定、管理。

IBM表示，z/OS 的第一個版本─V1R1，將會在本周五推出，至於第二
版V1R2則預定10月發表，V1R2將會內含一項新功能HiperSockets，可
允許硬碟分割區與其它分割區溝通，速度比正規電腦網路更為快速。

Gartner分析師Dale Vecchio表示，S/390被組織視為老舊應用軟體典
範的時間不會太久，透過結合開放產業標準，如 HTML、HTTP、XML、
J2EE技術，大型主機平台上的新企業應用軟體將成為可能，再加上整
合原有OS/390應用軟體，將可達到減少作業成本、降低複雜度。

為了增加大型主機競爭力，IBM 也對於大型主機軟體的收費作出改變
，從過去以總系統計算能力為基礎的方式，改以僅就軟體執行分割區
的計算能力收取費用。

早期G5、G6大型主機仍然可以執行z/OS，但無法使用只有新z900機器
能夠執行的功能，像是上述的64位元定址、HiperSockets、智慧型資
源目錄。

除了z/OS外，IBM於14日推出的新網路軟體─WebSphere，也是一套著
眼於大型主機用戶的軟體。IBM的WebSphere是能在z/OS、OS/390上執
行的軟體工具，提供對於企業Java技術，也就是J2EE的支援，讓軟體
開發者撰寫能在多種電腦系統執行的企業應用軟體。

■ 殺手級網站開發套件─PHP+PostgreSQL

沈美君/編譯
2001/03/27

Zend Technologies公司對於推廣PHP不遺餘力，並發展 PHP加值產品
、技術支援等；Great Bridge則是提供以PostgreSQL解決方案為主的
公司。兩家公司宣布未來將攜手合作，提供發展者以開放原始碼為基
礎的動態、資料庫導向網站的開發套件。

------

當PHP遇見PostgreSQL —前者是被公認最適合用來開發動態網頁的伺
服器端命令稿語言，後者則被視為開放原始碼中，最強大的資料庫管
理系統。推廣PHP的Zend Technologies，與提供PostgreSQL解決方案
的Great Bridge ，於26日宣布合作，未來將緊密整合 PostgreSQL與
 PHP ，提供發展者以開放原碼為基準的動態、資料庫導向網站的開
發套件。

這項搭售合作計畫，結合兩家公司的產品，可望為企業帶來可行的網
頁應用程式解決方案。Great Bridge與Zend的合作，範圍擴及各項產
品和服務，最大受益者莫過於使用者。Zend所推出的加值套裝產品，
將來會整體搭售於服務中，包括Zend IDE ﹙整合開發環境﹚、 Zend
 Cache ﹙提升網站效能、降低硬體 I/O 資料存取﹚、Zend Encoder 
Unlimited ﹙可編碼 PHP原始碼﹚，以及客製化的顧問服務。 Great 
Bridge則將提供經過品質測試、加強版的PostgreSQL，並推出相關技
術支援、顧問諮詢方面的專業服務。

除此之外，Great Bridge表示未來所釋出的PostgreSQL，將會與Zend
合作，推出與PHP 更緊密結合的安裝版本，兩家公司也表態，未來將
會陸續推出相關的整合開放原始碼計畫。

根據Netscraft的統計，目前全世界約有兩千五百萬個網站，使用PHP
作為網頁發展工具，這些網站中包括許多著名企業和線上出版網站，
例如思科、AuctionWatch、AltaVista、PC Magazine﹙德國﹚、Wall
 Street Online Germany。

對於此現象，Great Bridge總裁Robert Gilbert指出，「PHP 儼然成
為企業開發網頁應用程式的關鍵利器，而PostgreSQL的效能與可用性
也不下於商業等級的資料庫管理系統。」他表示，將兩者高度整合，
等於是把極具威力的解決方案，置於開發者的雙手上，協助開發者建
置企業等級的電子商務網站或網頁應用程式。

■ 美國國家安全局為Linux核心建立安全機制

沈美君/編譯
2001/03/26

Linux 在IT產業界的影響力不斷擴大，美國國家安全局也體認此趨勢
，並且計畫將安全局內所研究的安全架構和機制，移植到Linux 核心
中，欲將其發展成最不可能被駭客攻下的安全系統。

------

正當Linus Torvalds與其他核心發展者為下一版的核心2.5 規劃藍圖
時，美國國家安全局﹙US National Security Agency，NSA﹚的資訊
安全研究單位Peter Loscocco提出了可應用至Linux核心的mandatory
 access control ﹙代理存取控制，MAC﹚機制，不同於目前最常被
實作的discretionary access control﹙任意存取控制，DAC﹚。

在 DAC協定下，系統安全以「使用者」為管理單位，而非「被使用者
所執行的程序」，也就是說，若駭客取得系統管理者密碼，將可對作
業系統為所欲為。但MAC卻透過檢驗程序﹙process﹚的方式，監視並
確認可能對系統安全造成危害的工作，降低出錯的風險。

其實NSA早在1993年就開始發展MAC安全架構，1999 年的「Flask」即
為MAC的安全加強標準規格，並於同年夏天開始將Flask實作到 Linux
核心中，去年12月釋出第一版「SELinux」。由NSA 所開發的SELinux
，並非獨立的Linux套件，NSA希望SELinux能夠內建到核心中。

NSA表示將與專長於軟體模擬的 VAWare公司一起合作，在開放原始碼
的原則下，開發安全的 Linux系統平台。這項代號為「NetTop」的計
畫，結合VMware的虛擬機器技術﹙virtual machine technology﹚與
NSA 專長的密碼學，可在同一層網路上，設置不同等級的安全機制。

目前NSA 以「實體網路」的分區作法，來實作不同階等的安全機制，
藉以確保極機密和敏感資料的安全性。也就是說，如果有位NSA 的局
員擁有兩種不同的安全等級，他桌上就得放兩部電腦，分別銜接不同
安全等級的網路，使用不同的應用程式。

NSA資訊安全研究單位的指導 Paul Pittelli表示，「NSA成員對於現
成開發好的商業軟體有使用上的需求。」並且在使用這些套裝軟體時
，希望能夠不需要因為安全關係，而要求使用者在不同的電腦上使用
不同的軟體。「在安全考量上作加強的虛擬機器平台，可以協助我們
達到這樣的目標。」Pittelli指出，NetTop下一步將會朝向虛擬機器
上的安全實作來進行，強調平台上不同處理程序和安全等級的獨立性
與互不干擾的特性。這樣的安全機制，對著重保密及安全考量的政府
機關或企業公司，都是相當重要的。

■ 應用程式的萬國通行證ICU 1.8釋出

梁天佐/編譯
2001/03/26

應用軟體侷限於區域市場，不僅收益有限，開發成本也相對提高；軟
體業巨人微軟的成功有絕大部分即來自旗下軟體的成功國際化。開放
原始碼的Unicode國際元件ICU是經濟有效的國際化解決工具，讓應用
程式輕易轉換文字、貨幣、日期等格式，最新版ICU 1.8日前釋出。

------

Unicode國際元件ICU﹙The International Components for Unicode
﹚是一組C 與C++函式與資料庫工具，最新版ICU 1.8於日前公開開放
下載。

ICU 1.8 主要的功能改進部分為：增進語文校勘碼操作效能並使之相
容於Unicode Collation Algorithm ，應用程式介面轉換子支援統一
碼的所有編碼，改善ISO-2022轉換子﹙converter﹚並新增JIS、JIS7
、JIS8轉換子，IBM內碼轉換表更加完善，新增Unicode字串處理如大
小寫比對處理函數以及、 u_strrstr()、u_strtok_r()、u_memcpy()
等函數，

此外ICU 1.8 還新增了語文校勘﹙collator﹚、轉換子與重複中斷子
﹙break iterator﹚、新增函數 safeClone()能產生可在不同執行緒
中使用的類別物件，新增九種印歐語系音譯，改善阿拉伯字母字型，
以及更新語系資料庫等。ICU 1.8 主要提供：日曆支援、字集轉換、
語文校勘、日期格式、語文選擇、訊息分類與格式、標準化、數字與
通貨格式、時區、音譯標準與文章格式等地區化功能，完整且穩定，
並支援各種系統平台

應用程式轉換成地方版本時，除顯示字串訊息的翻譯，數字、日期、
通貨與語文格式的轉換之外，程式碼、資料庫與這些輸出格式的相依
性也必須更改，若每轉換一種地方版本就要重複同樣的工作，不僅成
本負擔過高，時效性差，也造成維護的困擾，因此應用程式國際化的
技術大多將程式中的字串、資料庫甚至某些特別的程式碼分離成獨立
的資源檔案，讓程式需要時才進行存取，同時更改變數格式與必要的
處理程序，使之不受語文格式影響。

ICU 即提供應用程式國際化必要的工具與操作，程序依次分別為：翻
譯字串、移除相依性、處理格式﹙數字、通貨、日期﹚、修改語文關
聯、使用相關字集、語文格式偵測、轉換非Unicode 內文與處理多語
文內容。應用程式經過ICU 的處理，便能輕易的轉換成各國地方版本
，大幅提昇效率。

ICU 是由一群公司與個人志願者所聯合推進的開放原始碼合作計劃，
因IBM主導大部分的工作，所以以IBM Public License註冊，由於IBM
國際化的工業實務經驗與常態性的支援， ICU得以不斷更新版本與增
加功能。

■ 玩GAME有一套  Transgaming讓Linux能玩Windows遊戲

張書銘/編譯
2001/03/26

目前在Linux上想玩Windows版本的遊戲，除了使用WINE模擬器來模擬
Windows環境外，就只能期待知名移植廠商 Loki軟體把好遊戲移植到
Linux上。不滿現況的Transgaming公司，將開發專屬的多媒體應用程
式介面，讓Linux上也能直接玩Windows的遊戲。

------

過去 Linux上由於缺乏良好的專屬的多媒體應用程式介面，因此在多
媒體應用，特別是在遊戲開發市場，顯得相當薄弱。現在由WINE模擬
器開發者組成的團隊，將為 Transgaming公司開發多媒體應用程式介
面，讓Windows遊戲能直接在Linux上執行。

Linux在多媒體領域上，始終落後微軟視窗，以遊戲產業而言，Linux
缺乏一個能和DirectX 多媒體應用程式設計介面相提並論的工具，因
此許多好遊戲都無法順利轉戰Linux 平台，玩家只能依賴像WINE一類
的Windows模擬器來望梅止渴。

為了解決這類問題，TransGaming 公司預備設計一套能與微軟視窗的
DirectX 完全相容的多媒體應用程式介面。這個介面以開放原始碼的
WINE為基礎，因此TransGaming 公司招募了許多從事WINE模擬器的開
發人員加入。除了能讓Windows遊戲都能在 Linux環境下100％相容，
也希望能藉此讓Linux能有與Windiws API相容的多媒體應用程式設計
介面，加速其他多媒體應用的發展。

Transgaming的執行長表示，目前這項開發計劃代號暫時定為WineX，
由於Transgaming為一家開放原始碼的開發公司，因此WinxX完成後也
會釋出原始碼。

■ Linux使用者社群自力推廣Linux

李孟諺/編譯
2001/03/26

義工是社會提昇的一股動力。在開放原始碼領域中，也有許多 Linux
義工默默地為Linux付出，希望將Linux推廣給每一個人。在美國華府
一年一度所舉辦的FOSE高科技電腦展中，就有Linux 使用者團體在沒
有企業的資助下，在會場設攤宣傳Linux。

------

FOSE﹙Federal Office Systems Exhibition﹚ 是在華府每年舉辦一
次的高科技電腦展，主要對象是聯邦政府、地方政府、軍事及私人資
訊工業公司的決策高層。會場中主要的產品多半是以非Linux 作業系
統的軟體為主。 去年有六家Linux公司參與FOSE電腦展，但今年只有
 Critical Linux 一家公司與會。不過，有許多 Linux使用者團體﹙
Linux User Group，LUG﹚，如 NoVaLUG與DCLUG，在沒有企業資助的
情況下，依舊獨力出資在會場中設攤，推廣Linux給電腦使用者。

展覽會場上，幾乎大多數的攤位都販售能在微軟視窗上操作的產品。
Linux 使用者社群認為，因為那些廠商都以利益為出發點。他們則是
出自於Linux社群的精神，在非Linux的電腦展中設攤，將 Linux推銷
給甚至連 Linux都沒聽過的人，並且讓電腦使用者了解 Linux的魔力
所在。對於Linux社群而言，因為出自於對Linux的熱忱而參加展覽，
並表示他們希望每年都能出席FOSE高科技電腦展。

Linux使用者社群對Linux大廠未能參與展覽深感遺憾，並表示 Linux
廠商不參與電腦展，是因為該電腦展並不是針對Linux 等開放原始碼
軟體所舉辦的。 Linux廠商並沒有做到如微軟當初大力推廣其視窗作
業系統時的努力，反而對其股價錙銖必究，而且眼中只有投資人與股
東。

Linux 使用者社群並指出，微軟公司經過多年努力，才讓視窗系統變
成個人電腦作業平台的主流，特別是視窗95推出時大受歡迎，代表著
親和力高，低技術性的作業系統才是大眾所期待的。只不過Linux 廠
商似乎沒有學到這一點，而把重點擺在如何成為伺服器與科學應用的
作業系統。至於將 Linux推廣給一般企業與家用使用者的工作，只能
落到 Linux使用者社群的身上。

Linux使用者社群並認為，許多報導有關Linux公司的新聞中，很容易
忽略像 Tim Bogart與Brian Bayus或Maury Merkin、Justin Swain、
Billy Ball等人。但是他們都是Linux核心程式的重要開發者。IBM、
VA Linux、紅帽公司等其他業者雖然都在開發Linux套件，不過Linux
 使用者社群才是Linux發展的真正核心。

NoVaLUG的發起人Tim Bogart表示，自從他接觸Linux後，就一直使用
Linux處理各種事務，至今已經兩年多了。他覺得有必要為Linux盡一
份力，所以成立了這個團體。 Brian Bayus則表示，現場有許多人參
觀過他們的攤位，並詢問有什麼產品可賣。不過他們都很驚訝與迷惑
，因為軟體都是免費贈送的。

在非Linux的展覽中，Linux使用者社群所設的攤位自然引人矚目。然
而在會場中最常被問到的問題是：誰擁有Linux。Linux使用者團體絕
對會回答：是我們大家。

   http://fose.com/generalinfo/faqs.html

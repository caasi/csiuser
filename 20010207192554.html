■ Transmeta將釋出Mobile Linux
謝良奇
2001/02/05

Transmeta計畫推出專供行動設備之用的Linux套件─Mobile Linux， 其計畫領導人正是現任Transmeta工程師的Linux之父托瓦茲。專供嵌 入式開發者應用於網際網路應用設備開發的Mobile Linux，特色是極 低的儲存空間需求。透過Mobile Linux對於嵌入式設備開發的強力支 援，將有助促進網際網路應用裝置更加普及。

Transmeta目前正計畫推出一套專為網際網路行動設備設計的Linux套 件─Mobile Linux，現為Transmeta工程師的Linux之父托瓦茲正是此 套件的開發領導人。Mobile Linux專為嵌入式設備等場合設計的特點 ，如低度的資源需求、模組化安裝等等，可降低在資源受限的設備上 開發Linux系統的困難。

本次LinuxWorld大會上，Transmeta預告將推出行動Linux套件Mobile Linux的消息。與其說是Linux套件，事實上Mobile Linux更像是嵌入 式Linux的開發工具。Mobile Linux 是Linux之父托瓦茲與Transmeta 工程師群數年來的開發成果。

提供行動設備開發者之用的Mobile Linux，首要特色就是極低的儲存 空間需求。透過其cramfs檔案系統的能力，附加glibc、程式庫及X視 窗系統的核心，僅佔 8MB。該項優點使儲存空間本就相當有限的嵌入 式設備，能提供多餘空間給Web瀏覽器或mp3播放程式等其它軟體。

此外，Mobile Linux 內含許多專為Transmeta硬體優化的功能，例如 對以該公司產品為基礎的設備提供完整驅動器支援、為Crusoe處理器 作的最佳化、對LongRun電源管理技術的加強等等。

僅管尚未正式問世，已有數家廠商推出基於Mobile Linux的產品，例 如Gateway Connected Touchpad。而諸如cramfs等Mobile Linux公開 的某些部份，早已納入許多PDA專用的Linux套件。倘若Mobile Linux 屆時推出能如預期般吸引嵌入式開發者的目光，將有助推動網際網路 應用設備普及的腳步。

■ LinuxWorld展覽會頒出多項大獎
李孟諺
2001/02/05

自上月底展開的LinuxWorld世界博覽會，一週以來提供許多 Linux以 及開放原始碼領域的新消息，為Linux以及開放原始碼的未來建構出 許多美麗的願景。如同一般博覽會，本月2日在 LinuxWorld的會場上 也頒發了多項大獎，給予發展Linux有功者一份肯定。

一群由工業專家、科技記者、分析師以及顧問所組成的評審委員會， 二日在LinuxWorld的會場上依據他們的票選結果頒出許多大獎。給予 為Linux與開放原始碼打拼的業者一個肯定。

藍色巨人IBM再度展現科技大老的魅力，以其Linux小型企業解決方案 贏得系統整合的首獎。這套解決方案包括了DB2、WebSphere伺服器應 用以及Lotus Domino。主要的功能是整合上述工具，提供完整的資料 整合、傳送協助客戶的工具訊息，以及網頁製作、資料管理等。

同樣獲得評審青睞的CA公司，以其推出號稱是電子化企業解決方案旗 艦版的Unicenter TNG 也獲得最佳系統整合方案。UniForum協會會長 ，也是評審之一的Alan Fedder表示Unicenter TNG 是Linux相當重要 的解決方案。一家公司透過該系統可以管控不同的部門、事業或生產 線。而且支援紅帽、SuSE、Caldera與TurboLinux版本的Linux。

贏得LinuxWorld最佳研發工具首獎的是Borland 軟體公司。他們獲獎 的產品是Kylix。Kylix系統是一個類似以Delphi開發的Windows 作業 平台，在Windows平台上用Delphi撰寫的應用程式，經由Kylix編譯後 可以使用在Linux平台上。Borland RAD產品部門副總裁Thornhill 表 示他們很榮幸Kylix 受到肯定。他們將會致力改善現有的視窗技術， 而且會盡速將Linux打入主流市場。

除了會場內優良場商的廠商受到肯定之外；在會場外，自由軟體基金 會﹙Free Software Foundation， FSF﹚也同樣的頒獎給為Linux 以 及開放原始碼努力的傑出人士。這次獲獎的人是在Mesa 3D 圖解運算 函式庫有傑出表現的Brian Paul。Mesa 3D 圖解運算函式庫是供給自 由軟體使用者使用與修改的軟體。Brian Paul也是第三位獲得自由軟 體基金會獎勵的傑出人士。

■ LynuxWorks協盟計劃 提升嵌入式Linux技術層次
謝良奇
2001/02/06

嵌入式設備市場越來越受到重視。LynuxWorks 旗下的BlueCat嵌入式 Linux，即將因Rappore Technologies、FairCom的產品技術移植，提 升在藍芽無線連接、資料庫解決方案的技術層次，可望滿足各領域的 嵌入式系統運用需要。

Rappore Technologies、FairCom兩家廠商分別於5日宣佈，將重要產 品移植到著名開放原始碼公司LynuxWorks旗下作業系統。專門從事可 靠性及安全無線軟體、服務供應的Rappore，將移植藍芽﹙Bluetooth ﹚技術至LynuxWorks的BlueCat嵌入式Linux。而另一家嵌入式資料庫 技術開發廠FairCom，將為LynuxWorks的BlueCat與另一套LynxOS即時 作業系統，移植c-tree Plus和FairCom資料庫伺服器的解決方案。

Rappore 進一步加入LynuxWorks的SynergyWorks協盟計劃後，將成為 LynuxWorks首家合作的藍芽技術廠商，意義相當重大。遵循藍芽 1.1 規格的Rappore 藍芽協定堆疊，將增強BlueCat嵌入式Linux的藍芽無 線功能，使採用嵌入式 Linux且具藍芽無線連接的嵌入式設備得以實 現。

LynuxWorks執行長兼主席Inder Singh認為，Rappore技術與經驗豐富 的工程組織，將對 BlueCat與專業服務組織有所助益。此外，提供與 Rappore合作、共同開發框架的 SynergyWorks協盟計劃，將可滿足各 OEM廠商對嵌入式Linux無線功能解決方案的市場需要。

另外一項關於LynuxWorks旗下BlueCat嵌入式Linux的技術移植，乃是 資料庫開發廠FairCom的嵌入式資料庫技術c-tree Plus檔案處理器， 以及FairCom資料庫伺服器的移植動作。FairCom的該資料庫解決方案 ，亦將移植至LynuxWorks的LynxOS即時作業系統。

c-tree Plus資料庫解決方案是內含超過一百個函式的程式庫，以C原 始碼釋出，提供資料插入、恢復、更新及刪除等所有資料庫功能，配 合FairCom資料庫伺服器，更能發揮主從架構具有的優點。FairCom此 項移植，將能為 BlueCat、LynxOS在航空通訊、自動化製造、網際網 路基礎設備產業的應用，帶來快速的資料儲存方案。

LynuxWorks市場副總裁表示，FairCom的c-tree Plus和資料庫伺服器 ，具備的快速生產力及卓越延展性及彈性，是 BlueCat、LynxOS相當 期待的合作對象。

■ Metro Link產品加BSD系統 強化嵌入式設備開發環境
張書銘
2001/02/05

以Micro-X Server技術縱橫嵌入式系統的Metro Link公司，為了推展 嵌入式設備開發平台的開發，決定將該公司的Metro-X以及Micro-X整 合到BSDi公司的eBSD系統內。藉著Metro Link的技術，eBSD系統使用 者也可以輕鬆加入嵌入式系統的開發行列。

Metro Link公司在日前宣佈將與BSDi公司合作，將該公司的 Metro-X 及 Micro-X兩套產品整合入BSDi公司的eBSD作業系統當中。這項合作 案將可為eBSD使用者提供一套完整的嵌入式系統開發環境，以及良好 的XServer操作環境。

Metro Link公司成立於1989年，專門發展嵌入式Linux和 UNIX作業系 統的圖形顯示技術及解決方案。而Micro-X與Metro-X就是其中最具代 表性的產品。在開發工程上，則可分別應用在設計與測試的功能。

Metro-X是一套強調可靠性以及易用性的X Server產品，它使用Metro Link公司的 ConfigX安裝技術，提供詳細的安裝說明與建議來幫助使 用者進行X Server的組態設定，讓缺乏經驗的使用者也能快速完成最 佳效能的安裝設定。同時，它也支援了目前最新的顯示卡以及顯示晶 片、並提供3D資料輸入的能力，對於多媒體相關程式的開發提供了良 好的支援。

Micro-X則堪稱目前市面上最小而美的X Server，整套 Micro-X 只有 570k。有不少的嵌入式設備都選用它做為操作環境，特別是IA﹙資訊 家電﹚產品。Micro-X，是一個標準的嵌入式X視窗，主要應用於跨平 台開發環境。精簡而又可跨平台的性質，使它能縱橫於追求「最小軟 體與電力」的嵌入式設備領域，而由於多數嵌入式設備都以它為操作 環境，因此它也可擔當系統開發完成後的測試重任。

■ NetBSD商業化推手─Wasabi
張書銘
2001/02/05

由NetBSD計畫主要成員所成立的Wasabi，採用開放原始碼模式為客戶 進行服務。Wasabi未來的營運方針是要讓NetBSD成為架構下一代網路 設備的基礎，進一步推動NetBSD商業化。

提供NetBSD系統服務的 Wasbai公司為了推廣BSD系統，除了在去年底 推出了NetBSD 1.5版外，也持續透過媒體與大眾討論關於 BSD系統的 各種資訊。這樣的做法，的確吸引了相當多原本對 BSD系統不甚了解 的使用者也加入了BSD系統一族。

Wasabi這家成立不久的新公司，是由NetBSD計畫主要成員所籌備的， 除了提供有關系統的各類技術支援服務，去年秋季也開始發售NetBSD 的光碟版本。Wasabi公司表示，NetBSD光碟版與網路上可取得的版本 並沒有太大不同，發售光碟的原因，是讓無法透過網路取得NetBSD使 用者也能順利安裝它。

事實上，NetBSD不是由Wasabi公司獨創的BSD系統，而是 NetBSD計畫 的產品。NetBSD計畫的目標，在於提供設計完整，穩定且快速的 BSD 系統、避免不必要的認證，及提供各種不同硬體架構都可執行的UNIX -like 系統。由於這項計畫是集合自願參加者來參與開發，因此它能 取得來自設計者最完整的原始碼以及廣大使用者的支援；同時相較於 其他 BSD系統，能夠應用的硬體架構也更多，除了支援i386架構外， 像Digital Alpha、Sun SPARC等也都可以使用NetBSD。

Wasabi公司也提到，未來營運方針將以持續提供新的整合性服務為主 軸。此外，未來Wasabi將與各網路設備領導廠商接洽，讓NetBSD成為 架構下一代網路設備的基礎。

■ 聽紅帽及IBM談GNOME的未來
沈美君
2001/02/07

LinuxPower.org日前分別採訪 IBM、紅帽、昇陽等公司，經由專訪可 得知他們對GNOME未來發展的期許，以及這些公司希望從GNOME的投資 上，得到什麼樣的回饋。看來，協助GNOME取得桌上型Linux市場的佔 有率，似乎是一致的目的。

KDE與GNOME 一直是在Linux圖形化視窗管理介面市場中，角逐榜首的 兩大競爭對手。LinuxPower.org日前以GNOME為主題，分別採訪IBM、 紅帽、昇陽等公司，對於 GNOME未來發展的看法，以及這些公司希望 從GNOME的投資上，得到什麼樣的回饋。

IBM的Linux技術中心主任Daniel Frye在接受採訪時表示，IBM在開放 原始碼社群投資約兩百名發展者，有些是全職為GNOME工作，IBM盼望 能加速GNOME在桌上型Linux平台的佔有率。而最早贊助 GNOME的第一 家商業公司紅帽，協助GNOME的目標與IBM大同小異，紅帽的實驗室﹙ Lab﹚經理David Mason指出，從一開始的孤軍奮鬥，到現在有這麼多 的廠商資助GNOME 發展，紅帽的角色可略作調整，將焦點放在架構的 問題上。

「例如，將注意力放在 GTK+2.0的發展，或者以我們的專業知識協助 Linux社群和我們的客戶」Mason 表示。作為全球首屈一指的Linux套 件廠商，紅帽的產品雖然大部分銷售給需要Linux 伺服器的企業，但 仍表明緊盯桌上型市場的走向。

目前IBM正與GNOME-print計畫小組密切合作，欲整合 GNOME與OMNI列 印架構，使其成為標準，這個計畫會將數以百計的列印驅動裝置引進 GNOME，包括具有照片品質的輸出裝置亦然。另外，IBM語音辨識系統 與GNOME和GTK的整合，也是正在進行的計畫。在開放原始碼計畫的執 照問題上，一般而言，Linux使用GPL、Apache是APL執照、而Mozilla 用MPL執照，若是IBM自己發起的開放原始碼計畫，則沿用IBM Public License執照。

在策略聯盟方面，IBM與Helix Code公司曾簽訂合約，以Helix GNOME 作為預設環境以搭售IBM的筆記型電腦，不過在IBM AIX上編譯GNOME 並不容易，相較於昇陽和HP的伺服器平台對GNOME的支援，IBM表示將 盡全力繼續發展；不過對於是否將GNOME作為預設桌面平台， Frye表 示，因為目前並無強烈的客戶需求，所以暫不作此打算。

Servertec 是一家專門提供網路開發與管理相關軟體、設施以及服務 的公司，該公司發表的Servertec Internet Server完全以Java 撰寫 ，能在不同平台之間存取與操作。其中使用的叢集技術可在網路上監 控大量遠端的設備。

Servertec Internet Server 是一款網路伺服器應用程式，體積小、 執行速度快且易於管理，能夠視需要調整，由於完全以Java撰寫程式 碼，不受作業平台限制。使用層除定位在一般網路開發應用之外，IA 製品及嵌入式系統也是其設定的目標。

為符合網路應用需求，Servertec Internet Server 支援多數如Java 、HTTP、SSL、與XML等開放標準與協定。此外，在安全性、執行效率 、存取控制、錯誤容忍、防止當機以及本身的擴充性上，都有一些特 別的設計與解決方案。軟體本身可以 Java、JSP、CGI、iScript改寫 ，以配合特殊需要。

專業版內含ISAPI，NSAPI，阿帕契，AOL伺服器，Java以及CGI閘道器 等元件；企業版除另含叢集技術元件外，所有專業版內含項目均以叢 集技術支援。企業或機構用戶僅需採購一次，在內部即可無限裝置， 無須為個別伺服器付費，也可視狀況決定是否選擇附有原始碼的版本 ，個人用戶則需以伺服器為單位向Servertec註冊。

另外，獨立軟體零售商註冊之後，就可以提供用戶各種內建版本，不 需另外支付版稅；所有版本均有一年維護與技術支援。Severtec在其 網站提供90天免費試用版，供人下載試用。

■ Intel對抗微軟的秘密武器─OSDL
沈美君
2001/02/05

開放原始碼發展實驗室﹙OSDL﹚於 1月24日在波特蘭正式成立，由包 括英特爾在內的十九家廠商資助，提供企業等級的高階硬體平台給開 放原始碼發展者測試軟體專案。英特爾宣佈支援OSDL，無疑是宣告與 微軟在策略上的對立。

英特爾於 1月24日宣布，集眾廠商之力投資兩千四百萬美元成立的開 放原始碼發展實驗室﹙OSDL﹚在波特蘭的Beaverton 正式開張，提供 企業等級的高階硬體平台研發適用於商務環境下的Linux 軟體，而此 目標市場正好與微軟視窗2000作業系統鎖定的市場重疊。

此外，開放原始碼發展實驗室成立目的是鼓勵軟體設計師在Linux 環 境下撰寫更多的應用程式，並利用實驗室所提供的高階環境來測試。 英特爾宣佈支援OSDL，無疑是宣告與微軟在策略上的對立。

英特爾副總裁William Swope 日前在LinuxWorld大會上表示，開放原 始碼的支持者應該更積極地投入軟體應用的發展，特別是研發適合企 業環境使用的技術，不要只滿足於在伺服器市場稱霸。

「抓住在企業環境中大顯身手的機會，為網路新經濟提供解決方案」 Swope 是這樣激勵開放原始碼社群的。另外Swope補充，OSDL 每隔六 個月就會提供兩萬五千美元作為獎勵，給對Linux Data Center 領域 最有貢獻的個人或團體。

OSDL甫成立就宣布開始支持兩個開放原始碼計畫的進行：其一為加強 Linux 支援十六個六十四位元處理器的能力，例如英特爾即將上市的 Itanium和McKinley處理器；其二為來自Jabber.org 的開放原始碼計 畫。

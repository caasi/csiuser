■ 支援Linux 2.4核心的Oracle 9i上市！

張書銘/編譯
2001/01/10

資料庫龍頭Oracle為了 Linux市場可說是卯足了全力，不但投入大量
資源開發適合 Linux的資料庫伺服器，現在更致力於研發應用程式伺
服器 Oracle 9i。在上週Linux最新核心2.4版釋出後，Oracle表示配
合最新版核心的Oracle 9i推出。

------

Oracle 9日宣佈能夠配合Linux 2.4核心版本的Oracle 8i以及9i正式
推出。在 Linux 2.4核心版本中，針對了許多企業用應用程式的需求
進行改良，加上Oracle的 Oracle 8i資料庫伺服器與9i應用程式伺服
器輔助，可謂如虎添翼。

Oracle 9i 採用了許多的新技術，並獲得多項專利。在基礎架構上，
它使用了Real Application叢集技術，以單一伺服器的設定來調整應
用程式的使用權限以及安全控管，而無須對用戶端電腦做太多麻煩的
設定，搭配Oracle 8i使用可大幅簡化管理程序。

在新版的 Oracle 9i中強調網路快取能力，以最新Surge Protection
技術增進網站的效能，幫助企業以低價硬體進行電子商務。SurgePro
tection的特色在於針對網路應用程式進行快取，過去的快取技術是
為快取靜態內容而設計，像是 JSP、ASP與CGI程式必須即時運算，快
取並不適合用於這些技術製作的網頁上，而Surge Protection則提供
安全檢查機制，在不影響伺服器與用戶電腦安全的條件下，提供運算
結果的快取結果，減少伺服器的運算工作及網路流量負擔。

由於資料庫市場競爭頗為激烈，Oracle為了對抗微軟等其他公司的產
品，自去年底開始企業網際網路服務，同時藉此服務，針對Oracle技
術網路會員釋出免費的Oracle 9i 應用程式伺服器。根據估計，目前
已有超過四十萬人次下載過相關產品。

■ Nokia與Lutris開發行動通信服務平台

馮文龍/編譯
2001/01/11

Nokia與Lutirs公司共同簽訂合作協議，Lutris將提供第一套Nokia的
行動通信平台上整合應用服務伺服器與開發工具的完整套件。

------

Lutris在8日與芬蘭電信設備製造大廠Nokia簽訂授權協議，將使開發
及推廣Nokia手機的應用服務更簡單快速。協議的內容包括了 Lutirs
將其Enhydra 3.5－Java/XML開放原始碼應用伺服器與Nokia的開發工
具套件一齊出貨。 Nokia開發工具套件包括WAP電話模擬軟體、Nokia
的WAP伺服器Active以及WAP工具套件等。

Nokia的Active伺服器為目前最被認可的企業用途的WAP伺服器，允許
關鍵性資訊從網站上傳遞到使用者手機上。WAP Phone 模擬程式與單
一用戶的Active伺服器連線，為使用Lutris的Enhydra 開發者提供了
一個模擬真實狀況的開發環境。

利用這套完整的開發工具，開發人員將能夠完整地存取Nokia Active
伺服器的全部功能。不論是程式碼編譯後的大小，或是利用WML1.1的
語法分析器除錯，都能利用這套工具完成任務，並能明顯地縮短開發
的時程。

Lutris的CEO Yancy Lind表示「針對企業的新線上即時接取技術已經
遠遠超過了傳統個人電腦的方式，而現在更增加了可攜式及手持式設
備的應用。」「與Nokia 的協議使得雙方技術得以整合，提供了一套
完成行動通信多通道通信最高境界的平台。」

Lutris同時開發了一套雙向的教學程式，著眼在加速Nokia WAP 平台
上的應用開發。Lutris並已成為Nokia 的WAP 開發夥伴的一員，使得
Lutris將能夠參與Nokia的開發訓練，並站在WAP應用開發的前線。

■ IBM、HP等電腦領導廠商響應Linux 2.4版核心

謝良奇/編譯
2001/01/10

近來開放原始碼界最熱騰騰的大事，莫過於Linux 2.4 版核心的釋出
，不但LinuxToday刊載Trovalds佈告的網頁閱讀人次，遠超過一般出
版物的最大新聞事件，連HP、IBM 等廠商亦相繼發言表示，將在包含
新版核心的套件問世後，測試自家產品以支援新版核心。

------

期盼已久的Linux核心2.4釋出後，立即受到熱切回應，自新版核心將
釋出的消息出現後，短短四天內，LinuxToday刊載有關Trovalds備忘
錄的網頁，已累積近十萬閱讀人次。除一般使用者外，企業領導廠商
們又是如何看待新版Linux核心，且聽聽如惠普、藍色巨人IBM等廠商
的發言。

HP的Bruce Perens表示，新版Linux核心可視為Linux跨進高階企業市
場的的一大步，主要的資料庫製造廠商將修改產品，以藉助新核心提
供磁碟存取的優點，這意味著磁碟存取表現將快於先前版本。他亦預
期一旦2.4版散佈套件正式推出，將受到HP開發者廣泛採用。

在進展快速超乎外界預料後，Linux 掀起的作業系統大戰已進入企業
系統這一個最後戰場。針對企業系統決定性的問題：延展性，在新版
Linux核心增添對稱多重處理器支援，增加系統延展能力。根據Bruce
Perens表示，HP將開發支援新版核心的機器，以運用經過改善對稱多
重處理器功能提供的延展性。

IBM的Linux技術中心主管Daniel Frye表示，IBM所有應用軟體將會在
2.4核心上作測試，當基於新版核心的套件測試版本產生，IBM將會在
那些套件上作應用軟體測試。IBM 將在未來四年投入50億元及1500位
開發者的強力支援，更有助掃除外界對 Linux的僅存疑慮。其次 IBM
已宣佈Linux將是旗下剛發表的zSeries 64 位元大型主機，採用的第
一套作業系統，甚至領先IBM自家的z/OS作業系統。

即使新版核心已正式釋出，要看到主要散佈套件包含2.4 版核心仍需
要一段時間。關於新套件何時問世，Red Hat 2.4核心開發者Preston
Brown 表示，Red Hat將會在準備妥當後推出新的版本。Red Hat計劃
在測試階段結束後，讓2.4核心成為RPM 套件，到時候Red Hat網路用
戶即可藉此升級現有的Red Hat 7.0系統到新版核心。

■ Netcraft：用PHP建立的網站超過500萬個

謝良奇/編譯
2001/01/11

PHP小組10日宣佈，根據Netcraft的調查，以PHP語言為基礎的網站總
數已經超過500 萬個，除了象徵此套開放原始碼手稿語言最新的發展
里程碑，更代表PHP語言當前的普及程度。PHP語言面對未來網際網路
上，更多的動態交易處理需求，將能提供其它工具、語言所不足的相
對優勢。

------

PHP 小組﹙The PHP Group﹚10日宣佈一項關於PHP語言的好消息，根
據來自Netcraft的一項調查指出，這套目前極普遍的開放原始碼網站
手稿語言﹙Scripting Language﹚，已建立超過500萬個網站，以PHP
語言在單月份即達15% 的高成長率，甚至超越網站本身的成長速率，
可看出PHP語言當前的蓬勃發展。

根據來自Netcraft的權威性調查，指出PHP於本月份達到建立510萬個
網站的象徵性指標，證實PHP乃是提供建立動態Web網站所需的最快速
成長技術之一。在現存靜態Web網站大批轉換至動態之際，將是PHP擅
長之處，

PHP 是一套伺服器端的手稿語言技術，能付予以往靜態網站缺乏的動
態能力，自1995年Rasmus Lerdorf開始發展以來，PHP 近來因應電子
商務需要而成長迅速。因認知PHP在高效能Web應用軟體的快速開發能
力，並受到以伺服器為基礎的應用軟體，近來爆炸性成長的推動下，
PHP逐漸以企業級規模的平台角色受到普遍接納。

除了相較於其它開發工具的易於使用、維護、更新等特性，PHP 提供
的快速開發、跨平台等優勢，更使得PHP在變化快速的Web開發領域中
，成為極理想的開發工具。難怪如 Restoration Hardware、Village
Voice Publishing等高流量網站，皆紛紛採用PHP建立Web網站，擔任
處理各自的關鍵性任務。

目前為Linuxcare開放原始碼專案工程師的 Rasmus Lerdorf表示，新
一代網際網路要求具動態處理的應用軟體，企業網站將需要更複雜功
能，PHP其專為Web設計的易用、可靠性、效能，能更為有效協助網站
開發，足可應付擁有數百萬網路用戶的專案。

■ Corel出賣Linux？23日將發表新營運策略

沈美君/編譯
2001/01/10

近年來營運績效不佳的Corel 公司23日將發表新營運策略，期望將公
司帶回可獲利的營運線上，策略可能包括賣出 Linux部門、重整原有
的繪圖及辦公室套件，以及與微軟有關的共同投資部分。

------

據分析師指出，Corel 高層主管將於23日，針對員工及媒體發表官方
新營運策略，計畫中可能包括將其未臻成熟的 Linux部門賣出，並設
定遠程目標，期望將公司帶回可獲利的營運線上。

由第三方管理顧問公司協助 Corel所做成的新營運策略，似乎略有提
振投資者信心的效果，Corel在NASDAQ的股價應聲上揚7.5個百分點。
Tera Capital公司技術分析師Duncan Stewart表示， Corel的新策略
可能包含三大部分：其一為針對 Corel Linux部門的銷售案，其二是
對於 Corel旗下原有的繪圖及辦公室套件重整，最後可能是與微軟有
關的共同投資部分。

微軟在去年10月投資Corel 將近一億三千五百萬美金，計畫協力發展
微軟的.NET計畫，但分析師們仍持續靜候這項投資案的細節。

以繪圖軟體CorelDRAW及辦公室套件WordPerfect起家的 Corel，從去
年開始發展Linux事業，但遲未有起色。過去九個月來，Corel原本預
期會有接近兩千萬美元的營收，但實際上只達成了六百一十萬美元。
傳聞Corel去年11月打算將 Linux 部門賣出，買主可能為紐約的一家
Global Linux Partners創投公司。

■ Sun對NFS在Linux上的發展深具信心

沈美君/編譯
2001/01/11

Sun打算如何推展在Linux上推廣NFS v.4呢？Solaris資料技術軟體工
程經理 Lee日前接受訪問時指出，NFS能夠滿足企業方面對Linux檔案
存取協定的需求，並且認為NFS足可成為Linux檔案分享的標準。

------

Solaris資料技術軟體工程經理 Lee指出，使用NFS將可為網路服務帶
來新契機。一個簡單的應用即為儲存服務提供者﹙storage services
 providers﹚可讓使用者直接處理遠端檔案，不需經過下載的程序；
此外，可增進大量儲存資料的使用效率，公司與公司之間可以更方便
的存取倉儲資料。

昇陽表示，在 Linux社群推廣NFS 所遭遇的最大挑戰，是社群認為昇
陽對NFS發展的主控權太大，為此，昇陽指出對NFS的研發態度將保持
「鬆手﹙ hands-off﹚」的姿態，並與密西根大學﹙ University of
Michigan﹚的資訊科技整合中心合作，開放社群的發展者可以自由加
入社群參與NFS V.4的開發，研發的指導工作將由IETF來擔任。

昇陽在去年2月已將此NFS 的核心部分元件「transport independent
 remote procedure call 」﹙獨立傳輸遠端程序呼叫，簡稱 TI-RPC
﹚釋出，可由昇陽官方網站下載，並從去年起就投注了雙倍資金以獨
立研發NFS V.4 在Linux 上的建置。而國際網路架構組織IETF，也有
意將昇陽的「NFS v.4 」網路檔案系統劃編為產業標準，以提升檔案
存取效率及各項應用。

與其他檔案系統相較之下， NFS被稱為是「真正利用網路特性來存取
檔案的協定」﹙file access protocol﹚，讓使用者能更容易地做大
量資源分享、網路檔案存取，在錯誤修復方面表現也更優異。若在傳
統一般的檔案系統下要存取遠端系統平台上的檔案，必須先下載到本
地端電腦上，NFS 的目的是直接以本地端電腦來操作遠端電腦上的檔
案。

■ 開放原始碼的SMS及WAP閘道器--Kannel 1.0

馮文龍/編譯
2001/01/10

想以最低成本跨入行動通信領域的人有福了！Kannel開放原始碼 SMS
及WAP 版本1.0正式發行，這是一套穩定性及擴充性高的WAP/SMS﹙簡
訊﹚伺服器，並通過驗證，擁有處理每秒數百個訊息的能力。

------

Kannel是一套穩定性及擴充性高的WAP/SMS ﹙簡訊﹚伺服器，並通過
驗證，擁有處理每秒數百個訊息的能力。Kannel的閘道器支援最普遍
的SMS中心協定，並由Wapit公司進行商業運轉，能夠扮演WAP及SMS閘
道器的功能。這使得未來開放原始碼社群跨足行動通信市場的門檻大
為降低。

蘭Tampere大學的獨立研究機構，Kannel被用在BioWAP，IMT的生物及
生物化學WAP服務。

導廠商，Wapit的產品系列能提供客戶完整的行動通信信息﹙message

Wapit 的客戶大多屬於經營行動通信服務的網路業者以及媒體公司，

以及標示語言如C-HTML、HTML、WML、XML。總公司位於芬蘭的赫爾辛

■ LinuxCare在澳洲推動開放原始碼社群會議

張書銘/編譯
2001/01/10

為了探討開放自由與商業利益的平衡點，並找出未來開放原始碼及免
費軟體的趨勢，LinuxCare公司將在澳洲舉辦linux.conf.au會議，邀
請社群與企業共同探討未來趨勢。議題中有一項有趣的觀點「 Linux
上的2D繪圖系統將成為主流，並出現不少殺手級軟體」在會議未開始
前就吸引了目光。

------

由於開放原始碼已逐漸與商業活動結合，如何在自由開放與維持利潤
之間取得平衡便成為重要的課題。為了讓社群人士與企業取得共識，
 LinuxCare 公司將於17日到20日，在澳洲新南威爾斯舉行linux.con
f.au會議，讓企業與社群討論商業活動對開放原始碼造成的影響。

事實上，除了 LinuxCare之外，新南威爾斯大學也是這項會議的重要
推動者之一。去年7 月，該校曾舉辦 Conference  for  Australian
Linux Users﹙CALU﹚活動，邀請許多Linux廠商討論產品開發趨勢。
獲得澳洲Linux使用者的好評，因此今年LinuxCare公司決定與該校合
作舉辦linux.conf.au會議。

此linux.conf.au 會議主要探討商業活動對開放原始碼的影響，因此
在議程中各廠商將以商業模式為討論主軸，像是KDE、Qt/Embedded等
都以探討公司商業模式為主題，而在技術方面則多半著重網路上的應
用，像是e-smith server and gateway探討簡化中小型伺服器的安裝
問題、Rproxy探討新的次級代理伺服器技術與瀏覽器整合技術等。

而 linux.conf.au會議中也將針對未來市場需求做分析。其中較為有
趣的觀點是 Linux上的2D繪圖系統將成為主流，並出現不少殺手級軟
體，因為目前3D幾乎已經成為主流技術，但議程中卻出現這樣與眾不
同的見解，在會議未開始前就吸引了大眾的注目。

   http://linuxpr.com/releases/3137.html
2、Linux.Conf.Au會議簡介
   http://linux.conf.au/

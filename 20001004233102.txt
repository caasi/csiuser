■ 微軟收購Corel股票 是否對Linux另有圖謀？

馮文龍/編譯
2000/10/03

微軟在本月一日突然出高價買下擁有不少Linux軟體的Corel公司，價
值一億三千五百萬美元的股票，雙方並在次日發佈所謂微軟.NET軟體
開發平台的策略聯盟消息。

------

本週一稍晚，市場傳出微軟買下了價值一億三千五百萬美元的 Corel
股票。未來兩家公司的聯盟將共同致力於.NET平台上相關新產品的發
展、測試與行銷，同時也將會有包括參加產品發表會、商業展覽，及
在彼此的網站上代言等行銷活動的結合。

此外，兩家公司並同意共同解決彼此在法律上的問題，Corel 的過渡
時期執行長Derek Burney透露，是關於Visual Basic在應用程式授權
上的問題，但微軟的開發部門主管Tom Button卻否認這項說法。

有趣的是，雙方的聲明都避免提到「Linux 」這個大家都認為敏感的
話題。Corel這家加拿大軟體公司，旗下擁有與微軟抗衡的Linux以及
辦公室套件WordPerfect，而WordPerfect 更是支援包括Linux等多種
平台，市場分析師就曾經推測微軟想要將其Office套件也移植其上，
但這點並未得到微軟的證實。

上週Corel的財務長便放出向外尋求資金的消息，所以此舉Corel的確
有被微軟「解放」的味道存在，不過雙方倒是興高采烈地宣告這項策
略聯盟，Corel執行長Burney表示，Corel非常高興宣佈與微軟的最新
發展，也相信這對Corel 長期的成長策略而言是重要的一步。

微軟的開發部主管Botton也表示，對雙方來說，這將是一個新世紀的
開始，微軟與Corel獲得了一個可以專注在.Net平台及服務的ISV，他
認為微軟的確投資了一家具有魅力的ISV。

■ SuSE Linux 7.0問世 創下歐洲軟體銷售第一紀錄

黃翔祺/編譯
2000/10/02

SuSE日前發行公司的新產品SuSE Linux 7.0 "Geeko"，這次所推出的產
品，所採用的核心程式為Linux2.4版，具有接近六百頁完整的說明書、
多項應用軟體、及超值個人咨詢服務，由於SuSE Linux 7.0成功的推出
，再加上 SuSE與其他如IBM等硬體廠商的合作，將使得SuSE一舉成為歐
洲最重要的軟體供應商。

------

歐洲聞名Linux廠商SuSE公司，日前發行公司的新產品 SuSE Linux 7.0
"Geeko"，這次SuSE共推出三個產品：SuSE Linux7.0個人版、升級版及
專業版。由於紅帽才剛推出該公司新版的紅帽Linux7.0，甫於八月推出
的SuSE Linux 7.0 格外受到歐洲廠商的注意。

SuSE這次推出新產品，立刻在市場上造成轟動，依據市場調查，SuSE在
推出第一天就賣了八萬四千套，而專業版則名列歐洲軟體銷售第一。由
於SuSE Linux7.0的推出成功，加上SuSE與其他如IBM等硬體廠商的合作
，將使得SuSE一舉成為歐洲最重要的軟體供應商。

在這次所推出的產品，所採用的核心程式為 Linux 2.4版，當然，依據
開放社群的慣例，在大多數的Linux 網站上，你也可選擇以下載的方式
來取得該產品。 

這次SuSE所推出的三項產品皆具有接近六百頁完整的說明書，其中個人
版產品在三張CD中，提供七百項應用程式，並提供六十天的個人諮詢服
務，而專業產品則在六張CD中提供超過一千九百項的應用程式﹙例如：
Java2開發程式、ALICE等重要開發產品﹚，並提供九十天的諮詢服務。
而升級版則是提供專業版的產品，但是在個人諮詢服務方面，則僅提供
六十天的諮詢服務。

SuSE Linux 7.0個人版產品價格為三十九點九五美元，專業版產品價格
為六十九點九五美元，如果過去為SuSE產品的使用者，可以購買SuSE所
推出的升級版產品，價格為四十九點九五美元。

沈美君/編譯
2000/10/03

開放原始碼解決方案提供者Kaivo，與Zope Application Server﹙應
用程式伺服器﹚的原創公司 Digital Creations，聯合宣佈將開班教

業工作的IT專業人員等。兩期的課程中，將由對Zope的入門引介、架
構，到開發實作、系統整合。

------

隨著Open Source 應用程式伺服器的需求增加，技術人員的訓練往往
是企業面臨的一大問題。開放原始碼解決方案提供者Kaivo ，有感於
此項服務的迫切需求，宣布將聯合Zope Application Server ﹙應用
程式伺服器﹚的原創公司 Digital Creations，於十月的第一個禮拜
，開班教授Zope相關課程，內容包含Zope的入門、架構簡介，到開發
實作、系統整合等層面。

Kaivo教育訓練副總裁Kathy Gosa指出，Zope 在開放原始碼社群中的
接受度越來越高，過去一年來，Zope被廣泛地整合到網站應用程式開
發及內容管理平台上，儼然已成為Web application server的第一選
擇。Gosa表示，高接受度後帶來的便是教育訓練的問題，如何指導發
展者能瞭解 Zope 的特性，再加以應用的現有的架構上，是企業面臨
的問題；而 Kaivo所做的事就是因應需求而提出的教育訓練服務。

此訓練的對象包括商業技術分析師、程式設計師，或是在企業工作的
IT專業人員等。上課操作環境將使用安裝Linux的 PC，並實際以Zope
來開發應用程式。Gosa表示，Kaivo在Open Source的教育訓練上已有
長時間的經驗累積，Kaivo 自認不只是單純提供訓練課程，而是具有
商業實作經驗的指導者，能夠使發展者將所學應用到企業的需求上。

Quintalinux 的CEO 補充說：「我們很樂意被授權在香港銷售紅旗的

■ Linux視窗介面程式設計又有生力軍─KBasic

張書銘/編譯
2000/10/02

Linux界最近在挖微軟牆角方面，顯得不遺餘力。繼 Phoenix Object
Basic這套號稱完全支援物件導向的Basic開發工具，以採用VB般視覺
介面吸引Visual Basic程式設計師後，現在，KDE上也推出一套Basic
開發工具─KBasic，而且挾著KDE 視窗介面設計的能力，聲勢不但較
Phoenix Object Basic毫不遜色，就連VB也將其視為大敵。

------

快速應用程式開發﹙ RAD﹚工具對於現在已經慣用視窗介面的使用者
與設計師而言，確實已經成為一種戒不掉的方便工具。因此對原本微
軟視窗系統上的程式開發者而言，想要轉設計UNIX/Linux應用程式，
可能得花上不少時間重新適應不同的開發環境。尤其是目前台灣地區
多數人都還在使用微軟產品的情況下，這個問題更使得不少科技人才
望而卻步。

也正因為這個困擾，KDE 的研發部門試圖在他們的視窗介面產品中加
上一套跟VB環境相似的程式開發軟體，這個想發所推演出來的就是最
新的KBasic。除了視覺設計介面以外，就連各種原件與控制項也和VB
相似，而且也擁有Phoenix Object Basic所強調的物件導向程式語言
特性，因此KBasic的發展格外令人注目。

和現有的Phoenix Object Basic比較，KBasic的元件多半都由自己發
展，因此第三方提供元件可能導致Phoenix Object Basic原始碼短期
內無法開放的問題，並不會出現在KBasic上，而且KDE原本在Linux視
窗介面上的開發能力也使他們對資料庫、網路及 GUI的支援上遠遠超
過 Phoenix Object Basic。

不過，由於KDE在計劃中已經排定明年夏天才推出KBasic 以及它的原
始碼，這與Phoenix Object Basic預估到明年第一季時就要釋出原始
碼來說，可能是一項不利因素。屆時，使用者會採用Phoenix Object
Basic，或是投入KBasic的懷抱？都還是未知數。

■ 運用開放原始碼Beowulf技術 高速電腦變便宜了

張書銘/編譯
2000/10/02

國際電算工程公司﹙ CEI﹚最近與 ASCI ﹙Accelerated Strategic 
Computing Initiative﹚簽訂合約，將致力於高速運算電腦的研發。
過去，雖然有許多大型廠商如 IBM等製造高速電腦，但始終因為成本
高昂而造成有需要的研究人員望之卻步。現在CEI將應用Beowulf技術
與開放原始碼兩項利器來解決這個問題。

------

過去幾年，全球高速計算主機情勢發生大變化，高速計算主機廠商數
目大幅減少，甚至連領導廠商如Cray及Convex，也分別被 SGI及惠普
併購。但美國能源部仍然繼續推動ASCI計畫，使高速計算主機的能力
迭創新高。

日本科技廳也投資四億五仟萬美元推動研發計畫，其中包含要在2002
年建造可達到40 TeraFlop 計算能力的主機；IBM 則宣布要斥資一億
美元推動藍色基因﹙Blue  Gene﹚計畫。這些都說明了高速電腦的市
場其實潛力無窮。

ASCI是由美國能源部所提出的計畫，目標是建置具有超強運算與模擬
功能的運作環境，用以確保美國核子武器之安全性及可靠度。因此，
美國能源部先後與許多廠商合作進行研究，希望能推出各種可靠的方
案，而在這一次與CEI簽訂的合約中，決定使用Beowulf叢集技術；並
採用CEI的EnSight Gold資料分析系統與紅帽的Red Hat Linux。

根據 CEI公司總裁表示，選擇這樣的搭配，是因為這種組合不但可以
提昇運算效率與穩定性，而且又節省軟硬體的開支。之前ASCI與其他
廠商的合約中，英特爾要價四千五百萬美元，IBM 合約價值九千三百
萬美元， Cray的合約金額據推測應介於英特爾與IBM的價格之間，而
決定使用Beowulf叢集技術與開放原始碼的CEI，則只需一百四十萬美

■ IDC預測：消費者市場將決定未來伺服器發展

沈美君/編譯
2000/10/02

IDC﹙International Data Corporation ﹚的分析師，預測未來十年
伺服器市場的趨勢和遠景。消費者力量抬頭，供應商的策略將起變化
，而競爭規則亦隨之主宰市場。除了頻寬、儲存裝置、處理效率的最
佳化，無線網路也是值得注意的方向。

------

IDC﹙International Data Corporation ﹚分析師Debra Goldfarb指
出，主宰未來十年伺服器市場的主要力量，將會是消費者市場。科技
對生活舒適度的幫助，例如語音辨識，在市場上會越來越重要，因此
，廠商與消費者間的關係，以及品牌口碑的建立，將會直接影響市場
佔有率，並改變競爭規則。

另一位分析師 Vernon Turner提到，未來的伺服器市場對於頻寬、儲
存設備和處理效率的速度要求會更高，以「兆位元組」﹙ terabytes
﹚為單位的儲存設備將會出現，企業除了有「伺服器部門」外，也可
能會新增另一「儲存設備部門」來作專門的處理。Turner也說明，無
線網路、點對點網絡也是值得注意的方向。

因為市場佔有率的提升，Linux、Windows NT、Windows 2000 、Unix
 在未來十年還會有一番激戰，而幾家領導廠商，如 IBM、Sun、 HP
、Compaq 和 Dell 將會聯手創造高達 65% 以上的伺服器市場。

IDC於1964年成立於美國麻省Framingham ，專注於觀察、研究全世界
快速發展的資訊及高科技產業，並提供顧客相關顧問服務，涉及領域
包含電腦主機、通訊、網路、周邊設備等市場研究及分析。

著肚子給我緊急支援。

，卻也不得不硬著頭皮上場。但上台之後，兩人臉上立刻堆滿笑容，

內容。 kelly拿著數位相機狂拍，我則在台下緊盯著整個活動過程，
網基科技執行長周文楷先生也到場為我們打氣。總算，活動順利結束
了，現場的反應十分不錯，但壓力一解除，我心中卻突然湧起一股莫
名的惆悵，只好藉著遊逛其他的攤位，希望能在人群中平撫自己的情
